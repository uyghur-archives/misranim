ھەممىدىن بۇرۇن  سىز كومپيۇتېرىڭىزدىكى مەشغۇلات سېستىمىسىغا ئاساسەن typescript تېلى غا كىرەكلىك بولغان دېتاللارنى تەييارلايسىز،ئەگەر سىز windows xp دە ياكى windows 7 دە بولسىڭىز
Visual Studio قىستۇرمىسنى قاچىلاڭ،ئەگەردە vistual studio نى كومپيۇتېرىڭىزغا قاچىلمىغان بولسىڭىز بۇيەردىن visual studio نى قاچىلاپ ئاندىن قىستۇرمىنى قاچىلايسىز
visua studio غا type script قىستۇرمىنى قاچىلاپ كود يىزىشنى باشلاش،باشقا تەھرىرلىگۈچ دېتاللىرىغا قارىغاندا ئىنتايىن ئازدا ۋە قولايلىق ،ئەگەردە سىزنىڭ باشقا مەشغۇلات سېستىمىڭىز بولسا visual studio نىمۇ قاچىلمىغان بولسىڭىز ئەنسىرمەڭ،بىز ئىشنى يىنىك دېتاللاردىن باشلايمىز
كىرەكلىك قوراللار
سىزگە پەقەت كىرەكلىك بولدىىغىنى بىر ياخشى تەھرىرلىگۈچ ،سىزگە تەۋىسيە قىلىدىغنىم sublime Text دېتالى،بۇ ھازىرقى ئەڭ مودا تەھرىرلىگۈچلەرنىڭ بىرى،تورپىروگىراممىلىرنى ئاچقۇچىلار ئۈچۈن كەم بولسا بولمايدىغان قورالدۇر.
submlime Text نى چۈشۈرۈش

ۋە بىر دانە توركۆرگۈچ ،توركۆرگۈچنىڭ ئەڭ ئىسلىنى تاللىشڭىز كىرەك،ھىچ بولمىغاندا Chrome ياكى firefoxنىڭ ھەرئىككىلىسنىڭ ئەڭ يىڭى نەشىرى بولسۇن
ئەگەر سىز تور پىروگىراممىلرىنى ئىچپ مەلۇم باسقۇچقا يەتكەن بولسىڭىز شۇنداقلاnpm بۇلاقلىغۇچىنىئىشلىتىپ باققان بولسىڭىز تۆۋەندىكى بىر قاتار بۇيرۇق ئارقىلىق

salam.ts نى قۇرۇپ بولغان بولسىڭىز بۇ ھۆججەتنىڭ ئىچىگە تۆۋەندىكى كودنى يىزىپ ساقلاڭ

كوماندا:
tsc salam.ts
