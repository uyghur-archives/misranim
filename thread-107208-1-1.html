Python دىگەن نىمە؟



Guido Van Rossum
پايتون يەنى(python)دەسلەپتە 1990 يىلى ھولىندا خەلېقئاراماتېماتېكا تەتېقىقات مەركىزى ۋە كومپيۇتېر تەتقىقات مەركىزدە Guido van Rossum تەرىپىدىن ئىجاد قىلىنغان  Python بولسا بىر ئىزاھلىق ھەرىكاتچان تېلدۇر،مەنبە كودى
ئۆزگەرگۈچى مىقدار ۋە مېتود پارامېتىرلارنى ئىلان قىلمايدۇ،بۇ ئوبىيكتىقا يۈزلەنگەن  ماسلىشچان تىل بولۇپ مەكتەپ ۋە ئۇنۋېرستىتلاردا ئىشلىتىلىۋاتىدۇ،بۇ تېل كودنىڭ ئوقۇلۇشچانلىقىنى ئاشۇرۇش ئۈچۈن لايھىلەنگەن بولۇپ توربىكەت ياساش ئۈچۈنلا ئىشلتىپلا قالماستىن يەنە ئۈستەل يۈزى پىروگىراممىلىرنى ئاچقىلى بولىدۇ،بۇ تېلنىڭ نورغۇنلىغان تەييار رامكىلىرى بولۇپ بۇلارنىڭ ئەڭ نوچىسى django,web2py ھىسابلىنىدۇ، بۇلاردىن پايدىلنىپ نەچچە مىنۇت ئىچىدىلا cms ئاساسىدىكى سانلىق مەلۇماتنى ساقلايدىغان بەت ياساپ چىقالايسىز،توردىن مەنبە ماتېريال ئىزدەشتە قينىلىپ قالمايسىز،بۇ تېلنى دۇنيادىكى ئەڭ داڭلىق     شېركەتلەر ئىشلتىۋاتىدۇ  جۈملدىن Eventbrite Quora Venmo FriendFeed Dropbox Google Duolingo قاتارلىقلار

Python نى باشلاش،ئاساسلىق گىرامماتىكىسى
Python نى باشلاش،ئاساسلىق گىرامماتىكىسىPython پىروگىرامما تېلىدا پىروگىرامما كودى ،كىڭەيتىلگەن نامى .py بولغان ھۆججەتنىڭ ئىچىدە ئىجرا بولىدىغان بولۇپ بۇنى shell يەنى كوماندا بۇيرۇق ئارقىلىق ئىجرا قىلىپ ئىچدىكى مەزمۇننى ئوقۇپ ئىلىشقا بولىدۇ،ئەلۋەتتە سىز خالىغان بىرنى تاللىۋالسىڭىز بولىدۇ،Python
نى قاچىلغان ۋاقىتتا Python كودلىرنى يازىدىغان ئەپچىل قورالمۇ تەڭ قاچىلىنىدۇ،بۇ قورالنىڭ ئىسمى IDLE (Python GUI) بولۇپ سىز pyton2.7 نۇسخسىنى قاچىلغان بولسىڭىز كومپۇتېرىڭىزنىڭ c:\python27 رايۇندىكى مۇندەرىجە ئىچدە ئورۇنلاشقان بولىدۇ...

Python نى Windows مۇھىتىدا قاچىلاش ۋە تەڭشەش


تۆۋەندىكى دەرسىك پايتوننى يڭىدىن باشلىغۇچىلار ئۈچۈن مۇھىم بولۇپ دەرسلىكدە Python نى توردىن چۈشۈرۈش windows ئۈچۈن ماس نەشرىنى تاللاش،قاچىلغاندىن كىيىن مۇھىت تەڭشەش ۋە Python نىڭ كود يىزىش قورالىدا تۇنجى ئەسسەلامۇ ئەلەيكۇم دىگەن بىرجۈملە پىروگىراممىنى يىزىش قاتارلىق مەشغۇلاتلارنى ئۆزئىچىگە ئالغان بولۇپ،يڭىدىن باشلىغۇچىلار چوقۇم بىسىپ ئۆتۈشكە تىگىشلىك بىرقەدەم ھىسابلىنىدۇ

Pythonدا Mysql بىلەن خىزمەت قىلىش

بۇ دەرسلىك Python دا Mysql بىلەن خىزمەت قىلىشقا مۇناسىۋەتلىك مەزمۇنلار ئۆز ئىچىگە ئالدىغان بولۇپ،جۈملىدىن Mysql غا ئۇلاش،Mysql دىن ئۇچۇرلارنى ئوقۇپ ئىلىش ۋە ئىجرا قىلىش شۇنداقلا Phpmyadmin ساندان قۇرۇلما شەكلى قاتارلىقلارنى ئۆزئىچىگە ئالدىغان بولۇپ Python دا مۇلازىمېتىر تەرەپ پىوروگىراممىلرىنى ئىچىش ئۈچۈن ناھايتى موھىم مەزمۇن ھىسابلىندۇ.

باشقا تېخىمۇ كۆپ پايتون ئۈگىنىش مەنبەلىرى ۋە ئۈگىنىش دەرسلىكلىرى ،شۇنداقلا پايتون توربىكەت ئىچىش رامكىلىرى توغرىسدىكى مەزمۇنلارنى تورماھىرى پايتون بىكىتدىن كۆرۈۋالسىڭىز بولىدۇ
تورماھىرى پايتون بىكىتى
ھازىرغىچە 1 ئادەم باھالىدىمۇنبەر پۇلى يىغىش سەۋەبى
matros1 + 10 يېڭىچە ئىزدىنىپسىز. ب.

ھەممە باھا نومۇرى : مۇنبەر پۇلى + 10   باھا خاتىرىسى
