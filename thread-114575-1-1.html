ئەسسالامۇئەلەيكۇم قىرىنداشلار

يىقىندىن بىرى ئۇيغۇرچە مۇنبەرلەرنى كۆپ ئارلايدىغان بولدۇم، قارىسام مۇشۇ Windows XP，Vista，7，8，8.1 مۇشۇ سېستىمىلارنى قاچىلاشنى باشتىن ئاخىرى تەپسىلى ئۆگىتىدىغان بىرە دەرسلىكنى كۆرمەپتىمەن (يا بار بولسىمۇ بەلكىم بايقىمىدىممۇ)، شۇنىڭ بىلەن بىر دەرسلىك ئىشلەپ قويۇش نىيتىگە كەلدىم(بۇنداق تىمىلار بار بولسىمۇ يەنە بىرنى يىزىپ قويسام ئارتۇقلۇق قىلماس)، ئاندىن تىرىشىپ-تىرمىشىپ تۆدەندىكى ۋىدېئونى ئىشلەپ قويدۇم.
بۇ مىنىڭ بىرىنجى قىتىم ۋىدېئو ئىشلىگەنلىكىم بولغاچقا بەك ئۇزاق ۋاقىت كەتتى. ئۆزى بىلگەن نەرسىلەرنى باشقىلارغا چۈشۈنۈشلۈك قىلىپ سۆزلەپ چۈشەندۈرۈشنىڭ ئۆزىمۇ بىر چوڭ قابىليەت ئىكەن. بۇرۇن ئانچە دىققەت قىلماپتىكەنمەن، بۇ قىتىمقى ۋىدېئونى ئىشلەش جەريانىدا ئۆزۈمنىڭ نۇتۇق سۆزلەش قابىلىيتىم شۇنچىلىك ناچار ئىكەنلىكىنى ھىس قىلدىم (بەك تەس بەك تەس ئىكەن دەرس سۆزلەش). شۇڭا بۇ دەرسلىك سەل چۇۋالچاق چىقىپ قالغاندەك، ھەم بەك ئۇزۇن بولۇپ كەتتى، تەخمىنەن ئىككى سائەتكە يىقىنغۇ دەيمەن، باشتا نىمە يادىمغا كەلسە شۇنى سۆزلەپ، ئاخىردا ئۆزگەرتىپ، ياساپ، بىر نەسىلەرنى قىلىپ مۇشىنچىلىك قىپتىمەن، ياخشى چىقمىغان بولسا زاڭلىق قىلماڭلار جۇما باللا(بولۇپمۇ سېستىما قاچىلاشنى ژىتىۋتىدىغان باللا ھەرگىز زاڭلىق قىلماڭلار مىنى)!
ئىككى سائەت نىمداق ئۇزۇن دەپ قىلىشىڭىز مۇمكىن، ھوللىۋۇدنىڭ نوچى كىنولىرىمۇ بۇنچىلىك ئۇزۇن ئەمەس، ھەي .....
لىكىن ھەر-بىر مەشغۇلاتقىچە ئىنىق، تەپسىلى بولغاچقا مۇشۇنداق ئۇزۇن چىكەتكەن ئوخشايدۇ، چۈنكى پۈتۈن تەييارلىق باسقۇچى ۋە سېستىمىنى قاچىلاش جەريانىمۇ تەخمىنەن بىرە يىرىم سائەتلەرگە بىرىپ قىلىشى مۇمكىن. شۇڭا ئۇخلاپ قالماي كۆرەسىلەر


بۇ دەرسلىك Windows 8.1 سېستىمىسىنى قاچىلاش دەرسلىكى بولۇپ، بىراق يۇقىردا دىگەن Windows XP，Vista，7，8，8.1 قاتارلىق سېستىمىلارنى قاچىلاشنىڭ ھەممىسىگە ئاساسى جەھەتتىن ماس كىلىدۇ. ۋىدېئو بەك ئۇزۇن بولغانلىقتىن ئىككى قىسىمغا بۆلىۋەتتىم، تەپسىلى مەزمۇنى تۆۋەندىكىلەرنى ئۆز ئىچىگە ئالىدۇ:
ئالدى بىلەنWindows 8.1 نى ياكى Windows xp，Vista，7，8 نىڭ iso فورماتىدىكى سېستىما بولىقىنى تەييارلاڭ.

Windows 8.1 نى چۈشۈرۈش ئادىرسىنى مەن تۆۋەندە تەمىنلەپ قويدۇم(مەن تەمىنلىگەن Windows8.1 سېستىما بولىقى ئېنگىلىسچە، ئورنۇتۇپ بولۇپ باشقا تىل بولىقىنى چۈشۈرسەڭلار بولىدۇ، ئۇيغۇرچىنى ياكى خەنزۇچىنى، دەرسلىكتە تەپسىلى مەشغۇلات ئۇسۇلى بار).

لىكىن ئاكتىپلاش نۇمۇرىنى ئۆزۈڭلار ھازىرلاڭلار.

(1) PE سېستىمىسىنى بارماق دىسكىسىغا قاچىلاش(بىرىنجى قىسىمدا).

(2) بارماق دىسكىدىكى PE سېستىمىسىدىن پايدىلنىپ، قاتتىق دىسكىنى رايۇنلارغا بۆلۈش(بىرىنجى قىسىمدا).

(3) PE سېستىمىسىدىكى UltraISOدىگەن يۇمشاق دىتالدىن پايدىلنىپ، Windows سېستىما بارماق دىسكا قوزغاتقۇچ ياساش (PE سېستىمىسىغا قىزىقمايدىغانلار بىۋاستە ۋىدېئونىڭ ئىككىنجى قىسمىدىن باشلاپ كۆرۈڭ، يەنى تۆۋەندىكى UltraISO دىگەن يۇمشاق دىتالنى چۈشۈرۈپ قاچىلاپ، ۋىدېئونىڭ ئىككىنجى قىسمىدىن Windows سېستىما بارماق دىسكا قوزغاتقۇچ ياساشنى ئۆگىنىڭ).

(4) Windows سېستىما بارماق دىسكا قوزغاتقۇچ ئارقىلىق Windows سېستىمىسى قاچىلاش(ئىككىنجى قىسىمدا).

(5) Windows 8/8.1 ئۈچۈن ئۇيغۇرچە تىل بولىقىنى چۈشۈرۈش(ئىككىنجى قىسىمدا).

(6) Windows 8/8.1 دىكى ئۇيغۇرچە خەتلەرنى چىرايلىق، گۈزەل، كۆركەم چىقىدىغان قىلىش(ئىككىنجى قىسىمدا).

(7) ئۇيغۇر كومپيۇتېر ئىلىمى جەمئىيىتى تەمىنلىگەن 86 خىل خەت شەكلىنى قاچىلاش(مەن ئەڭ ياقتۇردىغان UKIJ CJKخەت شەكلىمۇ بار)(ئىككىنجى قىسىمدا).

ئۇيغۇر كومپيۇتېر ئىلىمى جەمئىيىتى فونتلىرى

(8) ie，Chrome，Firefox قاتارلىق تور كۆرگۈچلەرنى ھەر قانداق تور بىكەتكە چىقسا ئۇيغۇرچىنى چىرايلىق چىقىدىغان قىلىش(ئىككىنجى قىسىمدا).
Google كەسمە يۈز رەسىمى

(9) QQ قورالىنى ئۇيغۇرچىنى تونۇيدىغان قىلىش

تۆۋەندىكىنى ئۇلۇنۇشتىن  Default_rdb چۈشۈرۈپ، پىرىستىن يىشىپ، QQ قاچىلغان جايدىكى
(يەنى C:\Program Files (x86)\Tencent\QQ\Resource.1.96.7979\Themesنىڭ ئىچىدىكى) Default.rdb ھۆججەتنىڭ ئورنىغا قويۇپ يەڭگۈشلىۋەتسىڭىزلام ئىش پۈتۈدۇ. يەڭگۈشلەشتىن بۇرۇن ئەسلىدىكى نۇسقىسىنى ساقلىۋىلىڭ، ناۋادا يەڭگۈشلەپ بولۇپ QQ نى قوزغىتالمىسىڭىز، ئەسلىدىكىسىنى قاچىلاپ قوزغۇتۇڭ. ھەم ۋىدېئودا سۆزلەنگەن ئۇسۇل ئارقىلىق،
通用包处理工具v3 قورالى بىلەن Default.rdb ھۆججەتنى تەھرىرلەپ، ئاندىن ئەسلىدىكىسىنىڭ ئورنىغا قويۇڭ.


(10) Photoshop بىلەن 绘声绘影 دا ئۇيغۇرچىنى قوللۇتۇش
