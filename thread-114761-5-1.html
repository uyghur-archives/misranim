غېرىپنىڭ زىندانغا سولۇنۇپ قاتتىق تاياق يەپ بىتاپ بولۇپ قالغانلىقىنى بىلگەن ( شىلەيلې ) دورا ساندۇقىنى ئېلىپ زىندان باقاۋۇللىرىغا نۇرغۇن كۆمۈشلەرنى ئارقا ئىشىك قىلىپ يۈرۈپ غېرىپ بىلەن كۆرۈشىدۇ ھەمدە ئۇنىڭ تاياق يىگەن جاراھەتلىرىنى ئەستايىدىللىق بىلەن دورا سۈرتۈپ داۋالايدۇ. غېرىپ ( شىلەيلې )نىڭ بۇ ياخشىلىنى ئۇنتۇپ قالمايدىغانلىقىنى ئېيتىپ كۆزلىرىگە ئىسسىق ياش ئالىدۇ. ( شىلەيلې ) غېرىپنىڭ كۆڭلىنى ياساپ، ئانىسىدىن شۇنداقلا سىڭلى گۈلجامالدىن خاتىرجەم بولۇشنى ئۆزىنىڭ ئۇلارنىڭ ھالىدىن ياخشى خەۋەر ئالىدىغانلىقىنى ئېيتىپ زىنداندىن قايتىپ چىقىدۇ. غېرىپ ( شىلەيلې )نىڭ كەينىدىن قارىغىنىچە ( ئىككىدىن ئايرىلالماسلىق ) ئىدىيەسىنىڭ نەقەدەر ئىللىقلىقىنى چوڭقۇر ھېس قىلىدۇ......
