قانداق قىلغان بىلەن ۋىندوۋسغا يەتمەيدۇ .كىمدەيدۇ ۋىرو ...

بۇ توغرىسىدىكى بىلىمىڭىز تولۇق بولمىسا قالايمىقان سۆزلىمەڭ، ۋىندوۋسنىڭ بىخەتەرلىكىنى ھازىر دۇنيا ئىتراپ قىلدى، بۇ گىپىڭىزنى كومپيۇتېر تىخنىكىسىدا ئالدىنقى قاتاردا تۇردىغان دۆلەتتىكى كىشىلەر ئاڭلىسا، سىزنى مەسخىرە قىلىدۇ.
ئەتراپىمىزدىكى بەزى كىشىلەر، Linux نى شۇنداق داڭلايدۇ، ھەممىدىن ياخشى دەپ، لىكىن نىمىشقا ھەممە ئادەم يەنىلام Windows نى ئىشلىتىدۇ دەپ سورايدۇ. ھەممە ئادەمنىڭ Windows نى ئىشلەتكىنى Windows نىڭ ياخشىلقىنى كۆرسىتىپ بەرمەيدۇ.
ھەي...ئىشلىتىپ باقمىغاندىكىن مىڭ چۈشەندۈرسەممۇ پايدىسى يوقتەك...بۇنى سۆزلەپ قالسام، Unix ئىچكى يادروسىنىڭ ئالاھىدىلكىدىن باشلاپ سۆزلەشكە توغرا كىلىدۇ...ھەممە ئادەمنىڭ Windows ئىشلەتكىنى بۇ پەقەت دۆلەت ئىچىدە كۆنۈپ قالغانلىق سەۋەبىدىن خالاس، پەقەتلام كۆنۈپ قالغانلىق.  Windows نىڭ بىرلام ۋاي دىگۈدەك ئالاھىدىلكى بار، ئۇ ئىشلىتىشكە ئەپلىك ۋە قولايلىق، يەنى مۇرەككەپ كومپيۇتېر بىلىملىرىنى تەلەپ قىلمايدۇ، شۇڭا ئۇ كۆپىنچە كىشىلەرنىڭ ئالقىشىغا ئىرىشتى. لىكىن باشقا تىخنىكىلىق، بىخەتەرلىك ۋە ھەر قايسى جەھەتتىكى ئالاھىدىلىكلەردىن ئىلىپ ئىيتقاندا Windows ئىچكى يادرونىڭ Unix ئىچكى يادروغا قارىغاندا قانچىلىك ئەخلەت ئىكەنلىكىنى بايقىغىلى بولىدۇ،  Linux نىڭ قۇرغۇچىسى Linux ئاتىسى Linus Torvalds ، مىكروسوفت ئىشلەپ چىقارغان بارلىق مەھسۇلاتلار ئىچىدە پەقەت Windows 7 گىلام خو بارمىقىنى كۆرسەتكەن...بۇنى ئۇچۇر تىخنىكىسى ساھەسىدىكى ھەممە داڭلىق كىشىلەر بىلىدۇ. پۈتۈن دۇنيادىكى سۈرئىتى ئەڭ تىز بولغان ئەڭ چوڭ 500 كومپيۇتېرنىڭ ئىچىدە 93.8% يەنى 469 غا Linux سىستېمىسى قاچىلانغان، ئالدىنقى 10 نىڭ ئىچىدىكىسىنىڭ ھەممىسىگە Linux قاچىلانغان.
Linux سىستېمىسى بىلەن MAC سېستىمىسىنىڭ ئىچكى يادرۇسى Unix دى كەلگەن، Android سىستېمىسىنىڭ ئىچكى يادروسىمۇ Linux دىن كەلگەن، بىز تۇرمۇشىمىزدا ئىشلىتىۋاتقان كىچىك تىپتىكى ئىلىكتىر ئەسۋابلىرى، مەسىلەن MP3، MP4، تىزگىنلىگۈچ دەمسىز ھەممىسىنىڭ سىستېمىسىنى Linux يادروسىدىن لازىملىغىنى ئىلىپ قالغىنى ئۆچۈرۈپ ئۆزگەرتىپ ئالغان. Google شىركىتى دۇنيادا ئەڭ كۆپ MAC كومپيۇتېرى ئىشلىتىدىغان شىركەتكە ئايلاندى، سەۋەبى MAC نىڭ بىخەتەرلىك نۇقتىسىدىن چىققان. يەنە مۇتلەق كۆپ ساندىكى تور مۇلازىمىتىرلارنىڭ ھەممىسى Linux ئىشلىتىدۇ. ئامىرىكا نۇرغۇن ھەربىي بازىللىرىدىكى كومپيۇتېرلىرىدىمۇ Linux قاچىلاقلىق، يىڭىدىغان چىقىۋاتقان ماشىنا ئادەم تىخنىكىدىمۇ Linux ئىشلىتىدۇ، ئاسمانغا قويۇپ بەرگەن ۋە يە ئاستىغا كىرىپ كىتىۋاتقان بارلىق تەكشۈرۈش ئاپىراتلىرى Linux ئىشلىتىدۇ.
Linux نىڭ بۇنداق تىز تەرەققىي قىلىشىدىكى سەۋەب ئۇنىڭ ئوچۇق كودلىقىدا، ھەممە ئادەم Linux تەرەققىياتى ئۈچۈن تۆھپە قوشالايدۇ، شۇڭا شۇنداق تىز تەرەققىي قىلىدۇ.
ئۇچۇر ساھەسىدىكىلەرنىڭ ھەممىسى مېكروسوفتنى تىللايدۇ، نىمىشقا دىسىڭىز، ئەگەر مېكروسوفتمۇ ئاشۇ سىستېمىنى ئوچۇق كودلۇق قىلىۋەتكەن بولسا، ھازىرقى ئۇچۇر تېخنىكىسىنىڭ تەرەققىياتى يەنە بۆلەكچە بولغان بولاتتى.
Linux نى ئىشلىتىش ئەمەلىيەتتە ھىچقانداق قولايسىزلىقلارنى ئىلىپ كەلمەيدۇ...بۇ پەقەت يۇقىرى تېخنىكىلىق مەھسۇلاتلارغا كۆنۈش ۋە كۆنەلمەسلىك سەۋەبى خالاس...كومپيۇتېر ساھەسىدە ئۇچۇر تېخنىكىسى توغرىسىدا خىلى بىلىمى بار ئادەملە بولسا، Linux ياكى MAC ئىشلىتىدۇ.
بەزىلەر يۇقارقىنى ئاڭلاپلا بىرىنجى بولاپ سورايدىغان سۇئالى، Linux دا چ چ ئىشلەمدۇ؟
ياق ئىشلىمەيدۇ، لىكىن Linux دا چ چ نىڭ ئىشلىمىگەنلىكى ھەرگىز Linux نىڭ خاتالىقى ئەمەس، بەلكى 腾讯 شىركىتىنىڭ كۆزى كىچىك ئىكەنلىكىنى بىلدۈرىدۇ(腾讯 نىڭ ئىنتايىن لاۋزا شىركەت ئىكەنلىكىنى ئۇنىڭ قۇرۇلۇشتىن بۇرۇن قىلغان ئىش-ئىزلىرىنى، تارىخىنى كۆرسىڭىزلام ئاندىن چۈشىنىسىز)، ئۇلار شۇ پايدا-مەنپەتنى، دۆلەت ئىچىدە Windows نىڭ ئادەتكە ئايلىنىپ قالغانلىقىغا قاراپلا، پۈتۈن كۈچىنى Windows دىكى يۇمشاق دىتاللارنى تەرەققىي قىلىشقا قاراتقان، 腾讯 نىڭ يەنە دۆلەت بىخەتەرلىك ئىدارىسى بىلەنمۇ كۈچلۈك ئالاقىسى بار، دۆلەت 腾讯 نى يۆلەيدۇ، شۇڭا ئۇ ھازىرقىدەك ھالەتكە يەتتى. QQ نىڭ يەنە بىر يوشۇرۇن ئىقتىدارى بار، ئۇ قىسقىغىنە ۋاقىت ئىچىدە پۈتۈن جۇڭگۇدىكى QQ ئورناتقان كومپيۇتېرلارنىڭ قاتتىق دىسكىسىكى ماتىرياللارنىڭ تىزىملىكىنى تەكشۈرۈپ چىقالايدۇ(مانا بۇ Windows نىڭ قانچىلىق بىخەتەر ئىكەنلىكىنى كۆرسىتىپ بىرەلەيدۇ)، بۇ ھەرگىز مىش مىش گەپ ئەمەس، قالغىنىنى ئۆزىڭىز پەرەز قىلىڭ، شۇڭا 腾讯 نىڭ مەھسۇلاتلىرىنى بەكمۇ ۋاي دەپ كەتمەڭ، QQ، 微信دىگەندەكلەرنى. ئەگەر Linux دا بەك QQ ئىشلەتكۈڭىز بولسا Web QQ نى ئىشلەتسىڭىز بولىدۇ.

مىنىڭ بۇنداق دىگىنىم Windows ياخشى ئەمەس دىگىنىم ئەمەس، ئۇنىڭ ئالاھىدىلكى ئۇنىڭ شۇنداق قولايلىق ۋە ئاددىيلىقىدا، باشقىلار بىر كۆرۈپلام ئىشلىتەلەيدۇ. شۇڭا ئادەتتىككى مەشغۇلاتلار بىلەنلام شۇغۇنلىدىغان ئورۇنلاردا Windows ماس كىلىدۇ.
ئىلىم-پەن، تېخنىكا، تەراققىيات، تور باشقا يۇقىرى تېخنىكىنى تەلەپ قىلىدىغان ئورۇنلارنىڭ ھەممىسىدە Linux ئىشلىتىدۇ. چەت ئەللەردىكى تەرەققىي تاپقان دۆلەتلەرنىڭ ئالىي مەكتەپلىرىنىڭ تەجىربىخانلىرىدىكى كومپيۇتېردا چوقۇم Linux بار. جوڭگۇدىكى قايسى بىر ئۇنىۋىرسىتىتنىڭ پروفىسسورى دۆلەت ئىچىدىكى ئالىي مەكتەپلەرنىڭ دەرسلىكىگە چوقۇم Linux نى ئالاھىدە دەرس قىلىپ قوشۇش لازىم، تەجىربىخانلىرىدىكى كومپيۇتېردا چوقۇم Linux بار بولۇشى لازىم دەپ ئوتتۇرغا قويغان، چۈنكى دۆلەت ئىچىدىكى ئۇچۇر تېخنىكىسىنىڭ تەرەققىياتىنى مۇشۇ كۆنۈپ قالغان Windows زور دەرىجىدە توسۇپ قويىدىكەن.

Unix ۋە C تىلىنى ئىجاد قىلغان Dennis Ritchie دىگەن كىشى 2011-يىلى 10-ئاينىڭ 9-كۈنى ۋاپات بولدى(ھەممىمىز بىلىدىغان Steven Paul Jobs  بولسا 10-ئاينىڭ 5-كۈنى ۋاپات بولغان)، لىكىن بۇ كىشىنى نۇرغۇن ئادەملەر تونۇمايدۇ. ئەگەر ئۇ كىشى بولمىغان بولسا ئاشۇ Steven Paul Jobs نىڭ شىركىتىنىڭ iPad،Iphone دىگەنلەر بولمايتتى، Linux بولمايتتى، Windows بولمايتتى، ھازىر بىز ئىشلىتىۋاتقان تېلېفۇن، ئولتۇرۋاتقان ماشىنا، ئايروپىلان، ئىشقىلىپ نۇرغۇنلىغان ئىلىكتىر ئەسۋابلىرى شۇ ئىپتىدائىي ھالىتىدە قالغان بولاتتى.(مۇشۇ سىستېمىلارغا مۇناسىۋەتلىك مۇھىم ئەرباپ بولغاچقا ئەسكەرتىپ ئۆتتۈم).

Linux نىڭ ئىشلىتىلشى مەلۇم دەرىجىدە كومپيۇتېر بىلىملىرىنى تەلەپ قىلغاچقا، بىزدەك ئادەتتىكى كىشىلەرنىڭ Linux نى بىمالال ئىشلىتىشى ئۈچۈن، Linux نىڭ نۇرغۇنلىغان ئۈستەل يۈزلۈك مەشغۇلات سىستېمىللىرى چىقتى، بۇلارنىڭ ئارىسىدا ئەڭ داڭلىقىدىن Ubuntu، Fedora، Mint، Red Het، Debian قاتارلىقلار بار.
ھازىرچە Ubuntu دىلام ئۇيغۇرچە تىل بولىقى مەۋجۇت.

ئەسلىدە Ubuntu بىز ئۇيغۇرلارغا ئەڭ ماس كىلىدىغان سىستېما ئىدى، لىكىن بىزمۇ دۆلەت ئىچىدىكى ئۇچۇر تىخنىكىسىنىڭ تەرەققىياتىغا كۆنۈپ Windows ۋە ئاشۇ QQ، 微信 لاردىن ئايرىلالمايدىغان بولۇپ قالدۇق.....

ئەگەر سىزمۇ Windows بىلەنلام چەكلىنىپ قالماي، Mac، Linux،Unix توغرىسىدا كۆپرەك بىلىم ئالسىڭىز، يۇقاردا قىلغان گەپلەرنىڭ تىگىگە يىتەلەيسىز...Linuxھەرگىز Windows قا يەتمەيدۇ دىگۈچىلەر، پەقەت كومپيۇتېر بىلىملىرىڭىزنىڭ تۆۋەنلىكىنى، Linux نى تولۇق چۈشەنمىگەنلىكىنى دىرەك بىرىدۇ. يۇقىرى تېخنىلىكىق مەھسۇلاتلىرىنى ئىشلىتىمەن دىگەنلام ئادەملەر ئىشلىتەلمەيدۇ... ئەتراپىڭىزدىكى كىشىلەرنىڭلام نىمە ئىش قىلىۋاتقىنىغا قارىماي، دۇنيادىكى كىشىلەرنىڭمۇ نىمە ئىش قىلىۋاتقىنىغا نەزەر سىلىپ قويۇڭ... بۇ يازمىنى ئاخىرىدا   kemter تەھرىرلىگەن. ۋاقتى  2014-1-11 11:17 PM
