ئانا تىلىمىزنىڭ خەلقئارالىق تەرەققىياتى ئۈچۈن بېلەت تاشلاڭ!



ئۇيغۇر تېخنىكا ساھەسىنىڭ باشلامچىسى، جۇڭگودىكى ئون مۇنەۋۋەر IT مۇتەخەسىسى، ئۇيغۇر سوفىت شىركىتىنىڭ قۇرغۇچىسى، شىنجاڭ ئۇنىۋېرسىتېتى ماتىماتېكا-سېستىما پەنلىرى ئىنىستىتۇتىنىڭ ئوقۇتقۇچىسى ئالىم ئەھەت ئەپەندىنى تونۇمايدىغانلار يوق. ئالدىنقى مەزگىللەردە ئالىم ئەھەت ئەپەندى باشچىلىقىدىكى ئۇيغۇر مىكروسوفىت شىركىتىنىڭ قان-تەرلىك جاپا مۇشەققەت بەدىلىگە كەلگەن Windows سېستىمىسغا ئۇيغۇرچە تىل بولىقىنى كىرگۈزۈش پىلانى ئەمەلگە ئىشىپ، Windows سېستىمىلىق كومپىيۇتېردا ئۇيغۇرچە خەت بولۇش ئەمەلگە ئاشقان ئىدى. بۇ شانلىق نەتىجىگە ئۇلىشىپ نۆۋەتتە Windows تېلېفۇنىغىمۇ ئۇيغۇرچە خەت بولىقى كىرگۈزۈشنى مىكرو سوفىت Windows شىركىتىگە ئىلتىماس قىلىنغان بولۇپ، بۇ ئىلتىماسنىڭ تەستىقلىنىش ياكى تەستىقلانماسلىقى بىلەت تاشلاش ئۇسۇلى ئارقىلىق ئەمەلگە ئاشىدىكەن.


ناۋادا بۇ ئارزۇيىمىز ئەمەلگە ئاشسا، ئۇيغۇر ئانا تىلىمىزنى نۆۋەتتە يەر شارىدا نوپۇزى بار Windows تېلېفۇنىغا كىرگۈزۈشتىن ئىبارەت بىر شانلىق نەتىجە بارلىققا كىلىدۇ. بىز ئەلۋەتتە ئۇيغۇر تېخنىكا ساھەيىمىزنىڭ تېخىمۇ گۈللىنىشى ئۈچۈن قولىمىزدىن كەلگەننى ئايىمايمىز.
تۆۋەندىكىسى بىلەت تاشلاش بىكەت ئادرىسى:http://windowsphone.uservoice.com/forums/101801-feature-suggestions
