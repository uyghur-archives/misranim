26-فىۋرال كەچ ،2013- 2014- پەسىللىك جۇڭگو  ۋاسكېتبول  بىرلەشمە مۇسابىقىسى  پەسىلدىن كىيىنكى تۇنجى ئايلانما باسقۇچىدىكى مۇسابىقىسى ئاخىرلاشتى. پەسىلدىن كىيىنكى يىرىم ھەل قىلغۇچ باسقۇچىغا كىرگەن تۆت كۇچلۇك كۇماندا تاللىنىپ بولدى.  گۇاڭدۇڭ كوماندىسى، شىنجاڭ كوماندىسى، دۇڭگۇەن كوماندىسى،  بىيجىڭ كوماندىسى تۆت كۈچلۈك كوماندا قاتارىغا كىردى.
  يىرىم ھەل قىلغۇچ مۇسابىقە 4-مارت رەسمىي باشلىنىدۇ.  تۆت كۈچلۈك مۇسابىقىسى ئىككى مۇسابىقە رايۇنىغا ئايرىلدى. ئالدىنقى مۇسابىقە رايۇنىدا گۇاڭدۇڭ كوماندىسى بىلەن بىيجىڭ كوماندىسى،  كىيىنكى مۇسابىقە رايۇنىدا شىنجاڭ گۇاڭخۇي كوماندىسى بىلەن دۇڭگۇەن كوماندىسى مۇسابىقىلىشىدۇ.
تۆت كۈچلۈك كوماندا مۇسابىقىسىدە بەش مەيدان مۇسابىقىلىشىپ، ئۇچ مەيدان ئۇتۇش تۈزۈمى قوللىنىلىدۇ، رەت تەرتىپتە ئالدىدا تۇرغان كۇماندا ئۆز مەيدانىدا بىر بىر قىتىم ئارتۇق مۇسابىقىلىشىش پۇرسىتىگە ئىرىشىدۇ (1-2-1-1).
  تۆت كۈچلۈك كوماندا  مۇسابىقە ئۇرۇنلاشتۇرۇشى
4-مارت (شەيشەنبە) كەچ ئۈرۈمچى ۋاختى سائەت 30؛5 دە بىيجىڭ كوماندىسى-گۇاڭدۇڭ كوماندىسى
5-مارت ( چارشەنبە ) كەچ ئۈرۈمچى ۋاختى سائەت 30؛5 دە دۇڭگۇەن كوماندىسى-شىنجاڭ كوماندىسى
6-مارت ( پەيشەنبە ) كەچ ئۈرۈمچى ۋاختى سائەت 30؛5 دە گۇاڭدۇڭ كوماندىسى-بىيجىڭ كوماندىسى
7-مارت  ( جۇمە ) كەچ ئۈرۈمچى ۋاختى سائەت 00؛6 دە شىنجاڭ كوماندىسى- دۇڭگۇەن كوماندىسى
8-مارت ( شەنبە ) كەچ ئۈرۈمچى ۋاختى سائەت 30؛5 دە گۇاڭدۇڭ كوماندىسى-بىيجىڭ كوماندىسى
9-مارت  ( يەكشەنبە ) كەچ ئۈرۈمچى ۋاختى سائەت 00؛6دە شىنجاڭ كوماندىسى- دۇڭگۇەن كوماندىسى
11-مارت (شەيشەنبە) كەچ ئۈرۈمچى ۋاختى سائەت 30؛5 دە بىيجىڭ كوماندىسى-گۇاڭدۇڭ كوماندىسى
12-مارت ( چارشەنبە ) كەچ ئۈرۈمچى ۋاختى سائەت 30؛5 دە دۇڭگۇەن كوماندىسى-شىنجاڭ كوماندىسى
13-مارت ( پەيشەنبە ) كەچ ئۈرۈمچى ۋاختى سائەت 30؛5 دە گۇاڭدۇڭ كوماندىسى-بىيجىڭ كوماندىسى
14-مارت  ( جۇمە ) كەچ ئۈرۈمچى ۋاختى سائەت 00؛6 دە شىنجاڭ كوماندىسى- دۇڭگۇەن كوماندىسى
