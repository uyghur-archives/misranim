مەن سۆيگەندەك سۆيمىدىڭ مېنى



سىمىلداپ يامغۇر ياغماقتا، ئەنە كۆردۈڭمۇ، كۈنلۈك كۈتۈرۈپ ئالدىمغا چىققىنى سەن ئەمەس... باشقا بىرى...
بۈگۈن  بىر كۈن سېنىڭ تېلغۇنۇڭنى كۈتتۈم، ساڭا تېلغۇن قىلدىم، مېنىڭ قۇلاق تۈۋۈمدە { دۈت ـ دۈت } قىلغان ئاۋازدىن باشقا ھېچقانداق ئاۋاز زاھىر بولمىدى...
سەن مەندىن { مەن سىزگە ئۆچ } دەپ بېقىگە دەپ سورىدىڭ، مەن جاۋاپسىزلا تېلغۇننى بېسىۋەتتىم، سېنىڭ ياندۇرۇپ تېلغۇن قىلىشلىرىڭغا بەرداشلىق بېرەلمەي تېلغۇننى ئىتىۋەتتىم، بىلمىدىڭمۇ؟ مەن ساڭا ئۆش ئەمەس، ساڭا يالغان ئېيىتقۇم يوق، سەن نىمىشقا مېنى { مەن ساڭا ئۆش } دىگۈزمەكچى بۇلىسەن؟
مەن يەنە ساڭا قاتتىق تەگدىم، ئەمەلىيەتتە ئۇنچىلىكمۇ ئەمەسقۇ؟ سەن مېنى ئىزلىمەس بولدۇڭ، ھەتتا تېلغۇنمۇ قىلىپ قويمىدىڭ، {سېنى سۈيىمەن، مەڭگۈ ساقلايمەن} دىگەنلىرىڭ نەگە كەتتى؟
مەن سېنى ساقلاۋاتىمەن، بىر قېتىم تېلغۇن قىلىشىڭنى كۈتۈۋاتىمەن، بىراق سەن شۇنداق جىمجىت...
مەن سېنىڭ { سىزنى سۈيىمەن } دىگەن سۈزۈڭگە ئىشەندىم، بىلەمسەن؟ ئىشىنىپ ئۈزۈمنى ئالدىدىم، چۈنكى مەن سېنى سۈيىمەن...

مەنبە غەمسىز
