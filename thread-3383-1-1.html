سالام دوسلار.....قانداقراق ئەھۋالىڭلار؟كۈندە مۇنبەردە ساق-سالامەت تۇرساڭلارمۇ ئەھۋالىڭلارنى سوراپ قۇياي،پاتىنىت ھوقۇقى ئالمىغان بولساممۇ خىلى قوشاقچى بوپ قالغان ئىدىم مۇنبەردە،لىكىنزە ئىشنىڭ جىقلىقىدىن،باشقا تىما بىلەن بولۇپ كىتىپ قوشاق يازمىغىلى خىلى كۈنلەر بولۇپ قاپتۇ،دىسەم-دىمىسەم سىغىنىپمۇ كەتمىدىڭلار مىنىڭ قوشاقلىرىمنى،مۇنبەردە مەندىن بۆلەك قوشاقچىلار جىق بولغاندىن كىيىن،شۇڭا مەنمۇ شىئىر يازاي،دەپ يىزىپ باقسام تاڭنۇر مەن بىلەن ئۇرشۇپ ئارام بەرمىدى،پاتىنتىمغا دەخلى -تەرۇز قىلدى،دەپ مىسرانغا چىقىپتىكەن،شۇڭا يەنە بۇ كەسپىمنى تىپىۋالدىم. ھەممىسى چاقچاق پاراڭ-ھە؟[s:6]

قارىسام نىمىشقىكىن مۇنبەرنىڭ كەيپىياتى سەل سۇسلاپ قاپتۇ،يا مەن شۇنداق ھىس قىلدىممۇ؟بۇنداق قىلىشىپ كەتمەيلى،بىردەم -يىرىمدەم مۇڭدۇشۇپ كۈلۈشۈپ ئولتۇرساق ۋاقتىڭلار باردۇ-ھە؟ [s:24]

ئەمىسا قوشاقنى باشلاپ قۇياي،قالغىنىنى سىلەر چۈشۈرسىلەر جۇمۇ؟ھە راست،بىر ئىشنى دەۋالاي،مەن قىنى؟مەن يوققۇ؟دەپ ئىنكاس يىزىشنىڭ ئورنىغا تۆت چاسا قىلىپ قاپىيدەش كەلگەن قوشاقتىن بىرنى يىزىپ قۇيساڭلا باشلاپ كىتىمىز-ھە؟[s:20]

ئۈچ....ئىككى...بىر.....باشلاندى!

توغرۇل ئاكىغا:

سالام سىزگە توغرۇلكا،
سىز مۇنبەردە بارمۇ-يوق؟
قوشاق يازدىم بىشىدا،
بىزگە جاۋاب بارمۇ-يوق؟

(توغرۇلكا،بەك ئالدىراشمۇ سىز مۇشۇ كۈنلەردە،ھىچ بىر قارىڭىزنى كۆرسەتمەيسىز،مۇنبەر ئۈچۈن تىنىمسىز ئىشلەپ جاپا چەكتىڭىز-ھە؟[s:2] )

مۇسلىنە ھەدىگە:
مۇسلىنەدە مۇنبەردە،
ساۋاق بىرىپ يۇرىيدۇ،
ھەر تىمىسى بەك ئۆتكۈر،
بىزگە تەلىم بىرىدۇ.

كىلىڭ ھەدە بىرلىكتە،
قۇشاق تۇقۇپ باقايلى،
يا ۋاقتىڭىز تەگمەسمۇ؟
قىنى كۈتۈپ باقايلى.

(ھەر بىر تىمىڭىز شۇنچىلىك ئۆتكۈر،ئاممىباپ،كۈچلۈك تەربىيىۋى ئەھمىيىتى بار،رەھمەت سىزگە[s:2] )

كۈنقىز غا:
كۈنقىز ئوماق دوستىمىز،
نىمە قىلىپ يۇرىيسىز؟
ئارقا بەتكە كىرۋىلىپ،
تىما مارلاپ يۈرەمسىز؟

(قىنى سىز -ھە؟ مەكىلىڭە،قوشاق يازىمىز.....قالغان تىمىنى مىسران تەستىقلىسۇن،سىز بىز بىلەن بىردەم ئوينىڭە[s:6] )

كەترەن گە:
مۇنبەرلەرنى كىزىپسىز،
بىزنى كۆرمەي يۇرىپسىز،
مەندىن قوشاقلار كەتتى،
قىنى جۋاب يازمامسىز؟

(كەترەن،نەلەردە يۇرىيسىز؟ما تىما ھازىرلا قىزىپ كىتىدۇ،ھىلى سىزنى تاپالماي قالماي،تىز كىلىڭە[s:14] )

شاتئاي غا:
شاتئاي تىنماي ۋىچىرلاپ،
كىزەر ھەركۈن مىسراننى،
قوشاق كەتتى كۆرمىدى،
مەن بىلىمەن بۇ قىزنى.

(دىسەم دىمىسەم ما قوشاقنى ھەممىنىڭ كەينىدە قىلىپ ئاندىن كۆرىدۇ[s:17] )

ئەلشىر گە:
بۇرۇن ھەركۈن كىزەتتى،
مۇنبەرلەردە سەيلى قىپ،
ئەمدى يوقاپلا كەتتى،
مىسرانىمغا باشلىق بوپ.

(ھەي،باشقۇرغۇچى بولسا ئاشۇنداق ئالدىراش بولۇپ كىتەمدىكەن؟بۇ ياققا كىرىپ تىز قۇشاق يىزىڭ-ھە؟سىزنى ساقلاتىمەن ئەمىسە[s:24] )

ئەركەم گە:
ئۇششۇق ئەركەم قىنى سىز،
يەنە نىمە قىلىسىز؟
مەن بىلەنلا سوقۇشۇپ،
ئۇرۇشاي دەپ ئويلامسىز؟

(ھەي ئەركەم،قىنى سىز-ھە؟ كىلىڭە ئەمدى ماۋۇ يەردە بىردەم قىزىرشىمىز[s:6] )

كەچمىشىم گە:
كەچمىشىم دەپ ئىسمىڭىز،
تارىخىمنى ئەسلەتتى،
بەكلا مۇڭلۇق شىئىرڭىز،
ماڭا يارنى ئەسلەتتى.

(شىئىرلىرىڭىزنى ياقتۇرىمەن،ھەقىچان ماڭا ئوخشاش قوشاقمۇ توقۇپ قۇيىسىز،كىلىڭە،بىللە بىردەم قوشاق تۇقۇيمىز[s:2] )

قەھۋە گە:
قەھۋە ئىكەن ئىسمىڭىز،
مۇنبەردە ساز گىپىڭىز،
ھەر ئىنكاستا كۆرىمەن،
تىما يازماس ئۆزىڭىز.

(سىزنىڭ تىمىڭىزنى پەقەتلا كۆرمىدىم جۇمۇ؟ئىنكاسنى بەك جىق يىزىپ كەتمىسىڭىزمۇ كۆرۈپ قالىمەن سىزنى،شۇڭا قوشاقتىلا دەۋالاي دىدىم گىپىمنى[s:2] )

سىگىنال غا:
ئەسلى ئىسىم ساراينى،
تاتقۇزۇپلا قۇيدۇقمۇ؟
بەلگە بولۇپ مۇنبەردە،
سىگىنال چىلىپ يۇردۇقمۇ؟

(چاقچاق-ھە؟ بۇنداق دىسە كۆڭلىڭىزگە كىلىپ قالمىسۇن يەنە ئاران تۇرغاندا،بۇ ئىسىممۇ ياخشىكەن[s:29] )

ئۇيغۇر تىكىن گە:
ئۇيغۇر تىكىن مۇنبەردە،
بەكمۇ ئۇزۇن تۇراتتى،
نىمە بولدى ئەمدى يوق،
خىلى ياخشى تۇراتتى.

(ھوي ئۇيغۇر تىكىن، قىنى سىز؟ ئۇنداق قىلماي مۇنبەرگە كىرىپ تۇرۇنە-ھە؟ما قوشاققا جاۋاب يازمىسىڭىز..ھە...قالغىنىنى دىمەي جۇما[s:24] )

تاڭنۇرغا:
شىئىر يىزىپ ھەركۈنى،
زىرىكمەيسىز نىمىشقا؟
شىئىرنىڭمۇ خۇمارى،
راس بار ئىكەن،شۇنداقما؟

(ھەر كۈنى بىر يىڭى شىئىرىڭىزنى كۆرمىسەم ئىچىم سىقىلىپ قالىدىغان بولدى ھازىر،يالغۇز مەنلا ئەمە ھەقىچان،قەلىمىڭىزگە بەركەت تىلەيمەن تاڭنۇر[s:2] )

كۆكنۇر غا:
كۆكنۇر سىزمۇ قوشاقچى،
كىلىڭ قوشاق يازايلى،
ئەزالارنى خۇش قىلىپ،
قىزىتايلى مۇنبەرنى.

(سىزنى ئۇزاق بولدى كۆرمىگىلى ئىشلىرىڭىز بەكلا ئالدىراشمۇ يە؟[s:10] )

سەمانى غا:
شىئىر _قوشاق ھەركۈنى،
يوللايتتىڭىز توختىماي،
ئىيتىڭ سىزگە نىم بولدى؟
بولدى ئەمدى سورىماي.

(سەمانى، سىز جىملا بولىۋالدىغان بولدىڭىز مۇشۇ ئاي-مۇشۇ كۈنلەردە،مانداق قىزىتىپ تۇرايلى مۇنبەرنى-ھە؟[s:6] )

ئەل قىزى غا:
توينى قىلىپ ئۇھ دەپلا،
خاتىرجەم بوپ يۇردۇقمۇ؟
توي قىلىڭلار دەپ قۇيۇپ،
دو تۇتقۇزۇپ كەتتىقمۇ؟
(سىزنىڭ ئۇ گىپىڭىز ئىسىمدە جۇمۇ تىخى،خەپ ئاز قالدىغۇ ماڭىمۇ[s:24] )

تۇمۇچۇققا:
تۇمۇچۇقتەك ئاۋازى،
ياڭرار ھەممە باغلاردا
ھە دىسىلا دىيىشەر،
مۇنبەردىكى ئىنكاستا.

(سىزنىڭ مۇنازىرە قىلغان ئىنكاسلىرىڭىزنى كۆرۈپ چىقىمەن،شۇڭا دەپ قۇيدۇم جۇمۇ؟[s:6] )

ئەمىسە مۇشۇنچىلىك يىزىپ تۇراي،قالغىنىنى سىزلەرگە قۇيۇپ قۇيدۇم-ھە؟
