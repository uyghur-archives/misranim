بىر جۈملىلىك كۈلدۈرگىلەر


بىرەيلەن تۇنجى قېتىم بازارغا چوكا مۇز ساتماقچى بولۇپ چىقىپ، توۋلاپ سېتىشنى خىجىللىق ھېس قىلىپ تۇرغىنىدا يېنىدىكى سېتىقچى ئاۋازىنى قويۇۋېتىپ«چوكا مۇز ساتىمەن» دەپ ۋارقىراشقا باشلاپتۇ، ئۇ شۇ ھامان «مەنمۇ شۇنداق» دەپ قوشۇپ قويۇپتۇ.

        %            %            %            %            % 
 

[align=justify]ئانىسى قىزىنىڭ ئوغۇل دوستىدىن «قىزىمغا كۈندە كىنو كۆرسىتىشتىن باشقىنى بىلمەمسىز؟» دەپ سوراپتۇ.
 

[align=justify]يىگىت ھەيران بولغان ھالدا؛ «سىزنىڭ دېمەكچى بولغىنىڭىز باشقا نېمىتى؟»دەپتۇ. 
 

%            %            %            %            % 

[align=justify]بىر جۈپ قۇشقاچ ئوۋچىنىڭ ئۇلارغا قارىتىپ ئوق ئۈزمەكچى بولۇپ تۇرغانلىقنى كۆرۈپ، بىرسى يەنە بىرسىگە؛ «سەن نەق مەيداننى قوغدا، مەن ساقچى چاقىرىپ كېلەي» دەپتۇ.
 

%            %            %            %            % 

[align=justify]ئىككى مەست ماشىنىنى ئۇچقاندەك ھەيدەۋېتىپ، بىرسى يەنە بىرسىگە؛ «ئېھتىيات قىل، ئالدىمىزدا جىددىي ئانلانما يول بار» دەپتۇ.

[align=justify]«نېمە، ماشىنىنى مەن ھەيداۋاتامدىم» دەپ جاۋاپ بېرىپتۇ يەنە بىرى.
 

 %            %            %            %            % 
 

[align=justify]بىر دېھقان ئۆزىنىڭ چىرايى سەترەك قىزىنى «قارانچۇق» ئورنىدا تۇرغۇزۇپ، قاغىلارنى قاچۇرۇش مەقسىتىدە، قوناقلىققا ئەۋەتىپتۇ، نەتىجىدە قاغىنى قاچۇرۇش مەقسىتىگە يېتىپلا قالماي، ئوغرىلاپ كېتىلگەن قوناقلىرىغىمۇ ئېرىشپتۇ.
 

%            %            %            %            % 

[align=justify]تۇغۇت بۆلىمىنىڭ بىر دوختۇرى ئۆز ئالدىغا ئامبۇلاتورىيە ئاچقان كۈنى ئاخشىمى ئايالى ئۇنىڭدىن؛
 

[align=justify]«بۈگۈن تىجارەت قانداقراق» دەپ سوراپتۇ، «خۇداغا شۈكرى يامان ئەمەس، ئانا-بالىنى قۇتقۇزالمىساقمۇ، دادىسى ساق تۇرۇپ بەردى» دەپتۇ دوختۇر.

%            %            %            %            % 

[align=justify]تىلى كەمتۈك بىرەيلەن ئاتسېنت بولغاندا، مەلۇم بىر ئوقۇغۇچىنىڭ باشقىلارنىڭ قەغىزىگە ئوغرىلىقچە قاراۋاتقانلىقىنى كۆرۈپ ئاچىققلانغان ھالدا؛ «سەن، سەن، سەن كۆچۈرۈشكە جۈرئەت قىلدىڭمۇ، ئورنىڭدىن تۇر» دېيىشىگە بەش نەپەر ئوقۇغۇچى بىراقلا ئورنىدىن تۇرۇشۇپتۇ. 

 %            %            %            %            % 

[align=justify]تۈنۈگۈن ئورۇقلىتىش چېنىقىشىغا تىزىملاتتىم. ئۇلارنىڭ تەلىپىچە بولغاندا مەشىق ۋاقتىدا كەڭرەك كىيىملەرنى كىيىش شەرت ئىكەن. نېمە دېگەن قاملاشمىغانلىق، كەڭرەك كىيىملەرنى تاپالىغان بولسام، ئورۇقلايمەن دەپ ئاۋارە بولاتتىممۇ».
