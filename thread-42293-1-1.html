MSN Messenger مىكروسوفىت شىركىتىنىڭ توردا ئالاقە قىلدىغان دۇنياۋى يۇمشاق دېتالى. ئىقتىدارى QQ غا ئوخشايدۇ بەزى ئىقتىدارلىرى QQ دىن ياخشى. تارىخىمۇ QQ دىن ئۇزۇن. 1999-يىلى 7-ئاينىڭ 22-كۈنى MSN Messenger رەسمى تارقىتىلغان. 2001-يىلى تارقىتىلغان Windows XP گە 4.6 Messenger نۇسخسى قۇشىۋېتىلگەن. ھازىرقى ئەڭ يېڭى نەشىرى 2011MSN Messenger ، Messenger نى ئىشلەتكۈچىلىرى ئۆزلۈكسىز كۆپىيىپ ھازىر 600 مىليوندىن ئاشقان. بۇنىڭ ئەڭ مۇھىم ئىقتىدارلىرىدىن بىرى ئۇيغۇرچە بىۋاستە كىرگۈزۈشنى قوللايدۇ.
مەن 2011 MSN Messenger ۋە 2009 MSN Messenger نى ئۇيغۇرچىلاشتۇردۇم. 2011 MSN Messenger نەشىرى Windows XP گە قاچىلانمايدىكەن. رايۇنىمىزدا Windows7 تۇللۇق ئومۇملىشىپ بولالمىغاچقا، Windows XP ۋە Windows7 لەردە ئورتاق ئىشلەيدىغان 2009 MSN Messenger نى قاچىلاش، ئۇيغۇرچىلاش ۋە ئىشلىتىش ئۇسۇلنى قىسقىچە تونۇشتۇرۇپ ئۆتىمەن. Messenger 2011 نىڭ ئاساسى مەشغۇلاتى مۇشۇنىڭغا ئوخشايدۇ.
1. MSN Messenger نى قاچىلاش
نورمال ئەھۋالدا ھەقىقى نۇسخسىدىكى Windows مەشغۇلات سىستېمىلىرىدا MSN Messenger بىللە قاچىلانغان بولىدۇ. ئەمما رايۇنىمىزدا كۆپ تارقالغان ساختا نەشىردىكى Windows ۋە Ghost ۋىندوۋىسلاردا MSN Messenger ئۆچۈرۋېتىلگەن بولىدىكەن. كومپىيۇتېردا Messenger بولمىسا تۆۋەندىكى ئادېرستىن بىۋاستە چۈشۈرۈپ قاچىلسىڭىز بولىدۇ.
http://wl.dlservice.microsoft.com/download/C/2/B/C2BF1A8F-AFC1-48CB-B04E-E3DEDAAA8DCF/zh-chs/wlsetup-web.exe
بۇ messenger 2009 نىڭ ئۆلچەملىك خەنزۇچە نۇسخىسى. توردىن چۈشۈرۈپ قاچىلاپ بولغاندىن كېيىن تۆۋەندىكى ئادېرسىتىن msnuy2009.exe دېگەن ھۆججەتنى چۈشۈرۈپ ئىجرا قىلدۇرسىڭىز ئۇيغۇرچىلاشتۇرۇش تاماملىندۇ.


چۈشۈرۈش 



1 سائەت ئالدىدا چىقىرىشچۈشۈرۈش (91.91 KB)
(2-رەسىم)
خەت نۇسخسى بەلگىلەيدىغان يېرىگە Alkatip خەت نۇسخسى قوشۇۋېتىلدى. ھەم يوقارقى ئۇيغۇرچىلاشتۇرۇش ھۆججىتنى ئىجرا قىلدۇرسا Alkatip خەت نۇسخسنى سىستېمىغا ئاپتۇماتىك قاچىلايدۇ.
2. ئىشلىتىش
ئىلگىرى تىزىملىتىپ ئېلىپ قويغان MSN ھېسابات ئىسمى بولسا يوقارقى 2-رەسىمدىكىدەك ھالەتتە بىرىنچى قۇرغا MSN ئې-خەت ئادېرسى، ئىككىنچى قۇرغا مەخپى شىفىرنى كىرگۈزۈپ بىۋاستە ئىشلىتىۋەرسىڭىز بولىدۇ.
MSN ھېساباتى بولمىسا تۆۋەندىكى ئۇسۇلدا ھەقسىز ئىلتىماس قىلىپ ئالسىڭىز بولىدۇ. ئىلتىماس قىلىش مۇۋاپىقىيەتلىك بولغاندىن كېيىن توردا باشقىلار بىلەن سۆزلەشكىلى بولىدىغان MSN Messenger ھىساباتى ۋە 25 گېگابايىت چوڭلۇقتا ئې-خەت بوشلۇقىغا ئېرىشىسىز.
2-رەسىمدىكى Windows Live ID بولمىسا يېڭىدىن تىزىملىتىڭ دېگەن خەتنى باسسىڭىز ئىلتىماس قىلىش بېتى ئېچىلدۇ.

مەن ئۇيغۇرچە ئىزاھات يېزىپ قويغان تەلەپ بويىچە كاتەكچىلەرنى تولدۇرۇپ، بەتنىڭ ئەڭ ئاستىغا قارىساق قوبۇل قىلىمەن ( 我接受) دەپ بىر كۇنۇپكا بار. شۇنى باسساق بولىدۇ. خاتالىق ئۇچۇرى چىقماي، كېيىنكى بەتكە ئالماشسا مۇۋاپىقىيەتلىك بولغان بولىدۇ. بىرەر كاتەك تەلەپ بويىچە تولدۇرۇلمىسا ماس خاتالىق ئۇچۇرى كۆرۈندۇ، شۇ تەلەپ بويىچە قايتا قىلساق بولىدۇ.
بۇنىڭدا QQ غا ئوخشاش شىفىر قوغداش... دېگەندەك مۇرەككەپ مەشغۇلاتلارنى قىلمىساقمۇ ھېسابات ئىسمى ، مەخپى شىفىر دېگەنلەرنىڭ ئوغىرلىنىشتىن ئەنسىرەشنىڭ ھاجىتى يوق. مەن ئىشلىتىۋاتقان ھىساباتىمنى ئالغىلى 6 يىلدىن ئاشتى، بىر قېتىممۇ شىفىر ئۆزگەرتىپ باقمىدىم.
Windows Live ID نى يېڭىدىن ئىلتىماس قىلىپ (ياكى بۇرۇنقىسنى ئىشلىتىپ) كىرگەندىن كېيىن ئىشلىتىشنى باشلىسىڭىز بولىدۇ.

(4-رەسىم)
توردا ئالاقە قىلدىغان يۇمشاق دېتاللاردا ئالاقە قىلدىغان ئادەم بولمىسا ئەھمىيىتى بولمايدۇ. شۇڭا بىزنىڭ ئالدى بىلەن قىلدىغان ئىشىمىز بۇ دېتالغا تورداش قوشۇش.
قارشى تەرەپنىڭ Messenger ئې-خەت ئادېرسنى بىلسەك 4-رەسىمدىكى تورداش قوشۇش تىزىملىكىنى باسىمىز.

(5-رەسىم)
5-رەسىمدىكى تەلەپ بويىچە قارشى تەرەپنىڭ ئادېرسنى كىرگۈزۈپ كېيىنكى قەدەمنى باسساق بولىدۇ. كومپىيۇتېردا ئالاقە قىلىش ئۈچۈن يانفۇن نومۇرى دېگەن تۈرنى تولدۇرمىساقمۇ بولىدۇ.

قارشى تەرەپنىڭ Messenger ئې-خەت ئادېرسنى بىلمىسەك 4-رەسىمدىكى تورداش ئىزدەش دېگەن خەتنى باسساق ئىزدەش بېتى ئېچىلدۇ. ئۇ يەردىن بىرەر ئاچقۇچلۇق سۆز ئارقىلىق ئىزدەپ تېپىلغان ئادەمنى قوشۇۋالساق بولىدۇ. مەسلەن: turk دەپ ئىزدەتسە ئىسمىغا تۈرك دەپ يېزىۋالغان بارلىق ئادەملەرنى تېپىپ بېرىدۇ.


باشقىلار بىلەن سۆزلىشىۋاتقاندىكى ھالەت
Windows XP مۇھىتىدا كومپىيۇتېرغا ئەلكاتىپ دېگەندەك بىرەر ئۆلچەملىك ئۇيغۇرچە كىرگۈزگۈچ قاچىلايسىز. ئاندىن word تا خەت باسقاندەك كىرگۈزگۈچنى ئۇيغۇرچىغا توغىرلاپ قويۇپ بىۋاستە بېسىۋەرسىڭىز بولىدۇ.
Windows 7 ياكى ۋىستادا ئايرىم ئۇيغۇرچە كىرگۈزگۈچ قاچىلاش ھاجەتسىز. ئۇيغۇرچە كىرگۈزگۈچ ۋىندوۋىسنىڭ ئۆزىگە قوشۇۋېتىلگەن.
يەنە، ئۇيغۇرچە قوللايدىغان يانفۇندىمۇ بىۋاستە ئۇيغۇرچە يېزىپ ئالاقە قىلغىلى بولىدۇ. تۆۋەندىكىسى Windows Mobile6.5 سىستېمىلىق يانفۇنغا ئۇيغۇرچە تونۇتقاندىن كېيىنكى سىناق ئېلىپ بارغان كۆرۈنۈش.




كەمچىلىكى: QQ دا قارشى تەرەپكە بىۋاستە ھۆججەت يوللىغاندا ياكى ھۆججەت قۇبۇل قىلغاندا بىۋاستە كومپىيۇتېر-كومپىيۇتېر شەكىلدە ئۇلىندىكەن. ئەمما ماۋۇ MSN Messenger دا كومپىيۇتېر-مۇلازىمېتىر-كومپىيۇتېر شەكىلدە ئۇلىندىكەن، ئوتتۇرىدا بىر باسقۇچ كۆپ بولغاچقا ھەم مۇلازىمېتىرى چەتئەلدە بولغاچقا بىۋاستە ھۆججەت، رەسىم يوللاش تېزلىكى QQ غا قارىغاندا سەل ئاستا.
<P>MSN Messenger نىڭ ئۇيغۇرچىسىدا ئىملا خاتالىقلىرىنى بايقىساڭلار ۋە باشقا تەكلىپ-پىكىرلەر بولسا ئالىپ تورىدىكى ئەسلى تېمىغا ئىنكاس شەكىلدە قالدۇرۇپ قويساڭلار بولىدۇ.
