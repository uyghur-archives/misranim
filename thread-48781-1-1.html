مۇنبەردىكى چەتئەل ناخشىللىرى ياختۇرىدىغان بالىلارنىڭ ھۇزىرىغا چەتئەلنىڭ ھەرساھەدىكى داڭلىق شەخىسلىرى كۈيلەنگەن بىر قىسىم داڭلىق ناخشىلارنى يوللىدىم ياختۇردىغانلار ئاڭلاپ بېقىڭلار ،


1 - ئەنگىلىيە مەلىكىسى { دىئاننا } كۈيلەنگەن ناخشا .،

ناخشا ئىسمى { دىئاننا }  ئۇقۇغۇچى : بىريان ئادامىس

2- داڭلىق ئوسكار كىنو خانىشى { ئېلزابىت تايلۇر } كۈيلەنگەن ناخشا ،

ئاخشا ئىسمى { لىبىرىيە قىزى } ئۇقۇغۇچى : مايكۇل جېكسون ،

3 - ۋوللىۋود چولپانى { نىكولاس كەيچى } كۈيلەنگەن ناخشا ،  

ناخشا ئىسمى { ئاخېرغىچە سۈيۈش } ئۇقۇغۇچى : تىرىسشا يېرۋود،  { ئاسمان تۈرمىسى } فېلىنىڭ ئاساسلىق ناخشىسى قىلىپ بىكىتىلگەن ،



4 - داڭلىق مودا ناخشا شاھى { مايكۇل جېكسون}  كۈيلەنگەن ناخشا ،

ناخشا ئىسمى {  سىنى  مەڭگۈ سۈيىمىز مودا ناخشا شاھى  } ئۇقۇغۇچى : كىۋېس فىران .
