cnbeta تورىنىڭ 6-ئاينىڭ 4-كۈنىدىكى خەۋىرى، ھازىرغىچە نۇرغۇن قېتىم Windows 8 نىڭ كۆرۈنمە يۈزىنى كۆردۇق، سىزدە مائۇسنىڭ ئەتىسىگە نىسبەتەن غەم-ئەندىشە بولدىمۇ؟ ئەلۋەتتە، 2017-يىلى 7-ئايدا ئېلان قىلىنىدىغان Mac ئۈستەل كومپيۇتېرى ئۈچۈن كۆپ چېكىتلىك كونترول قىلىش تاختىسى Magic Trackpad نى لايىھىلىگەن ۋاقىتتا، مائۇسقا بىر پەشۋا ئېتىلىدۇ. ئۇ ۋاقىتتا نۇرغۇن كىشىلەر ئالما شىركىتىنىڭ بازار قىممىتىنى تىلغا ئېلىشقا ئەرزىمەيدۇ، دەپ قارايدۇ، چۈنكى Magic Trackpad نىڭ تەسىرى ئانچە چوڭ بولۇپ كەتمەيدۇ. ئەمما ئۇلار مۇنداق بىر مەسىلىنى تونۇپ يەتمەيدۇ، ئۇ بولسىمۇ، بىز ھازىر بىزنىڭ تېگىشمە يۈزلۈك ئەسلىھە دەۋرىگە قاراپ مېڭىۋاتقىنىمىزدۇر. ئەمما ئەقلىيفۇن ۋە تاختا كومپيۇتېرلاردا مودا بولۇۋاتقان  تېگىشمە يۈزلۈك ئىقتىدارلار ئەنئەنىۋى كومپيۇتېرلارغىمۇ كۆچىدۇ.


ئەمما، نۇرغۇنلىغان جاھىل ئۇنسۇرلار بۇ يۈزلىنىشنى خالىماسلىقى، PC ئۈچۈن مەڭگۈ مائۇس بولمىسا بولمايدۇ، دەپ تۇرۇۋېلىشى مۇمكىن. ھازىردىن قارىغاندا، بۇ خىل كۆز قاراش پۇت تىرەپ تۇرالمايدۇ. چۈنكى، ناھايىتى تېز ياكى كېلەر يىلىلا مىكروسوفت بۇ ساھەگە چوڭ ئۆزگىرىش ئېلىپ كېلىدۇ. Windows 8 نىڭ ئېلان قىلىنىشىغا ئەگىشىپ، PC مۇ ئۆزىنىڭ مائۇس بىلەن ئۆتكەن رىيازەتلىك كۈنلىرىگە خاتىمە بېرىدۇ.
مىكروسوفتنىڭ بايانىغا ئاساسلانغاندا، Windows 8 ئېنىقلا مائۇستىن ۋاز كەچمەيدىكەن، ئۇ سېزىمچان ئېكرانلىق ئەسلىھەلەرنى قوللىغاندىن باشقا يەنە ئابونتلارنىڭ مائۇس ئارقىلىق مەشغۇلات قىلىشىنىمۇ قوللايدىكەن. ئەپسۇسلىنارلىقى شۇكى، Windows 8 دە مائۇس ئىشلىتىش بىر كۈلكىلىك ئىش بولۇپ قېلىشى مۇمكىن.
ئۇنىڭدىن باشقا، ئالما شىركىتى يېقىنقى كۈنلەردە ئېلان قىلغان يېڭى بىر ئەۋلاد مەشغۇلات سىستېمىسى Mac OS X Lion مۇ سېزىمچان تېگىشمە يۈزلىشىشىنىڭ ئومۇملىشىشىنى ئىلگىرى سۈرىدۇ.
مائۇسنىڭ جېنىنى ھەقىقەتەن چىڭ دېيىشكە بولىدۇ، ئۇ ئۇزۇن يىللاردىن بىرى ئىزچىل تۈردە كومپيۇتېر بىلەن بىرلىكتە مەۋجۇت بولۇپ كەلدى. مائۇس ھەقىقەتەن ياخشى، لېكىن ئۇنىڭ ئاخىرقى كۈنلىرى يېقىنلاپ قالدى.
