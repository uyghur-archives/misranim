Windows 7 نىڭ ئەسىلدىكى ئۇيغۇرچە خەتلىرى XP نىڭ ئۇيغۇرچە خەتلىرىگە قارىغاندا ياخشىراق بولسىمۇ، تازا چىرايلىق ئەمەس. XP  نىڭ ئۇيغۇرچە خېتىنى چىرايلىق قىلىش ئۈچۈن ماۋۇ تېمىدىكى ئامالدىن پايدىلىنىپ، سىستېما فونتلىرىنى مەجبۇرى ئۇيغۇرچە فونتقا ئالماشتۇرۋەتكەن ئىدۇق. Windows 7 گە ئۇ چارە ئاقمايدىكەن. ھەم سىستېما فونتىغا ئۆزگەرتىش كىرگۈزۈش قائىدىسزلىك بولۇپ، ئامالسىزلىقتىن بولغان ئىش.
Windows 7 گە سىستېما فونتىغا چېقىلماي تۇرۇپ ئۇيغۇرچە خەتنى چىرايلىق كۆرسىتىشنى نىسبىي ھەل قىلغىلى بولىدىكەن. تۆۋەندىكى رەسىملەرنى كۆرۈڭلار.







قانداق بولامدىكەن؟ سىلەرمۇ كومپىيۇتېرىڭلاردىكى Windows 7  نىڭ ئۇيغۇرچە خەتلىرىنى ئاۋۇ رەسىمدىكىدەك چىرايلىق قىلماقچى بولساڭلار تۆۋەندىكىدەك قىلىڭلار.
ئاۋۋال ماۋۇ يەردىن ukij tuz tom نى چۈشۈرۈپ قاچىلاڭلار.
http://www.ukij.org/fonts/fonts/UKIJTuT.ttf

UKIJTuT.ttf نى مائۇس ئوڭ كۇنۇپكىسىدا چېكىپ 安装 نى باسسا ياكى c:\windows\fonts قا كۆچۈرۈپ قويسا بولىدۇ. ئەلۋەتتە، ukij tuz tom ئىلگىرى قاچىلاغلىق بولسا بۇ مەشغۇلاتنى قىلمىساڭلارمۇ بولىدۇ.
ئەمدى بۇ ئادېرستىكى alip.themepack دېگەن ھۆججەتنى چۈشۈرۈپ ئىجرا قىلدۇرساڭلارلا ئىش تامام . ئېكران رەسىملىرى  NASA تارتقان ئالەم بوشلۇقىنىڭ رەسىملىرىگە ئالماشسا مۇۋاپىقىيەتلىك بولغان بولىدۇ. (پەخىرلىك ئالىمىمىز ئەركىن سىق ئەپەندى ياسىغان تىلىسكوپتا تارتىلغان رەسىملەرمۇ تېخى )

بۇ ئۇسۇلنى ئىشلەتمەي، ئەلكاتىپ 5.6 نى قاچىلاپ، كومپىيۇتېرنى قايتا قوزغىتىپ، ئېكران خاسلىقىنى تەڭشەيدىغان يەردىن تىزىملىك قاتارلىق ھەر قايسى يەرلەرنىڭ فونتىنى بىر-بىرلەپ ئەلكاتىپقا ئۆزگەرتىپ چىقسىمۇ بولىدىكەن. بۇ مەن ياسىغان تېما ھۆججىتىنى قاچىلاشقا قارىغاندا سەل ئاۋارىچىلىقكەن.
يەنە باشقا ئۇسۇلنى بىلدىغانلار بولسا مەرھەمەت...
