ئەسسالامۇ-ئەلەيكۇم، مۆھتەرەم مۇنبەر ئەھلى،كومپىيوتېر ئۇستىكارلىرى،تۆۋەندىكى قىينچىلقىمنى ھەل قىلشا ياردەم بەرسەڭلار!   يېقىندا پېقىرنىڭ كومپىيوتېرى ئېغىرلىشىپ كەتكەنلىك تۈپەيلىWINDOWS-ghost XP 2011 ھەقلىق نەشىردىكى  سېستىما پىلاستىنكىسىنى سېتىۋىلىپ، كومپىيوتېرىمنىڭ سېستىمىسىنى يىڭلغان ئىدىم،دېمىسىمۇ بۇ يىڭى سېستىمنىڭ نۇرغۇغ ئۆزگىچىلكى باربولغاندىن ئۈنۇم،سۈرئىتىمۇ ھەققەتەن  سۆيۇنگۈدەك دەرىجىدە ئىكەن.شۇ مەمنۇنلۇقۇم بىلەن ئىلگىرى-ئاخىرى بولوپ يىگىرمىگە يېقىن دېتال،پىروگىراممىلارنى قاچىلاپتىمەن،ئىككى كۈندىن كىيىن  تورغا چىقاي دېسەم   ئېكىراندا    (  宽带连接  )    نىڭ سۈرىتى يوق. شۇنىڭ بىلەن     (   开始   )   غا كىرىپ ،ئۇنڭدىن   (    设置   )   ئارقىلىق   (  网路连接   )   نى ئاچسام بۇيەردە پەقەتلا    (    本地连接   )    تۇرىدۇ، دېمەك، يىڭى سېستىما قاچىلغاندا  بۇ   (    宽带连接   )  دېگەن تۈر قۇرۇلمىغان ئىكەن. شۇنچە ھەپىلشىپ يىڭىدىن    (    创建一个新的网络连接     )  قۇرۇپمۇ باقتىم ئەمما بىريەرلىرىگە كەلگەندە ئەپلەشتۇرەلمەي توختاپ قالدىم، شۇڭا كەسىپ ئەھللىرىنىڭ مېنىڭ بۇ قىينچىلقىمنى ھەل قىلشقا ياردەم قىلشنى سەمميىلىك بىلەن سورايمەن.[table=100%,#ffff00]قايىل قىلارلىق، چۈشىنشلىك جاۋاپ بەرگەنلەرنى يازمام بەدىلگە كەلگەن  مەلۇم مىقداردىكى مۇنبەر پۇلۇم  بىلەن تارتۇقلايمەن
