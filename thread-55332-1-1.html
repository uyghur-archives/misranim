نۇرغۇن تورداشلار كومپيۇتېرىغا پارول بېكىتىۋېلىپ، ئۆزىنىڭ شەخسى ئۇچۇرلىرىنى قوغداشنى ياخشى كۆرىدۇ. گەرچە ئاز بىر قىسىم كومپيۇتېرلاردا Windows مەشغۇلات سېستىمىسىغا بارماق ئىزى بىلەن تىزىملىتىپ كىرىش ئىقتىدارى تەمىنلەنگەن بولسىمۇ، كۆپ قىسىم كومپيۇتېرلاردا يەنىلا پارولنى قولدا كىرگۈزۈشكە مەجبۇر بولىمىز، ناۋادا سىزنىڭ پارولىڭىز ھەم ئۇزۇن ھەم ئەستە تۇتۇش قىيىن بولغان پارول بولسا، بۇ Windows قا تىزىملىتىپ كىرىشتە يەنىلا ئاز بولمىغان ۋاقىتنى ئالىدۇ. ئۇنداقتا كۆپچىلىك ئادەم چىرايى ئۇچۇرلىرىنى تونۇش ئارقىلىق Windows قا تىزىملىتىپ كىرىشنى ئويلاپ باقتىمىكىن؟

Luxand Blink بولسا دەل مۇشۇ خىلدىكى ئېھتىياجنى قاندۇرىدىغان يۇمشاق دېتال! سىزدە چوقۇم بىردانە كامېرا بولۇشى ۋە سېستىما قوزغىلىشتىن بۇرۇن كومپيۇتېرغا ئۇلانغان بولۇشى كېرەك(كامېرا بولمىسا ئېكرانغا قاراپ ئولتاغان بىلەن چىرايىڭىزنى تونۇپ بەرمەيدۇ). Luxand Blink نى قاچىلىغاندىن كېيىن، ئۇ سېستىما قوزغىلىشتىن بۇرۇن بۇرۇن ئاكتىپ ھالەتكە كېلىپ، Windows نىڭ كىرىش كۆرۈنمە يۈزىنى كونتروللىقىغا ئالىدۇ، ئاندىن قاملاشقان چىرايڭىزنى كامېراغا تۇتۇپ بەرسىڭىزلا كۇپايە!
تورداشلار تەمىنلىگەن سىناق نەتىجىسىگە ئاساسلانغاندا، بۇ خىلدىكى Windows قا كىرىش ئۇسۇلىنىڭ ئۈنۈمى ناھايىتى قالتىس بولۇپ، كامېرا ئېچىلغاندىن كېيىن، كۆزىتىش رامكىسىغا چىرايىمىزنى ئۇدۇللاپ، كۆزىمىزنى يېنىكلا قىسىپ قويساق(Luxand Blink نىڭ Windows قا كىرىشتىكى سىگنال بەلگىسى دەل كۆزنى قىسىش)، سېستىمىغا ئۇدۇللا كىرىپ كېتىدىكەن.
Windows تىكى ھەر بىر ئابونتقا نىسبەتەن، بىر دانە ماس ھالدىكى چىراي ئۇچۇرلىرىنى تەمىنلەش زۆرۈر بولىدۇ، يەنى سىز Luxand Blink نى قاچىلىغاندىن كېيىن، سىز بىر دانە چىراي ئىپادە ئۇچۇر قېلىپىنى كىرگۈزۈشكە ئېھتىياجلىق بولىسىز، ئۇنىڭدىن كېيىن Win + L كۇنۇپكىسى ئارقىلىق بۇ ئىپتىدائىي چىراي ئىپادە تونۇش ئۇسۇلىدىن ھوزۇرلىنىشنى باشلىسىڭىز بولىدۇ.

ئەسكەرتىش: بەزى تورداشلار بۇ دېتالنىڭ كومپىيوتۇر سېستىمىسى XP ھالەتتە نورمال ئىشلىمەيدىغانلىقىنى ئىنكاس قىلدى. (شۇنداقتىمۇ ئۆزىڭىز بىر سىناپ بېقىڭ ). ئەمما Win 7 مۇھىتىدا تامامەن نورمال ئىشلەيدىكەن. شۇنىڭ ئۈچۈن بۇنىڭغا دېققەت قىلىشىڭلارنى ئۈمىد قىلىمەن.
