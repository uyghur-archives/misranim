مۇنبەرلەر نامىدىن پۈتۈلگەن شېئىر

(ئىزدىنىش) يىڭىلىق يارىتىش يولى،

شۇ (رىتىم )ئاستىدا ماڭىمىز ھامان.

(ئەل كۈيى )ياڭرايدۇ ھەربىر قەلىبتە،

(ئەل خۇشى)ھەردىلدا بىربۈيۈك ئارمان.

(ئانا تۇپراق)ئۇلۇغ ئانامدەك  ماڭا،

ئۇ مىنىڭ مەڭگۈلۈك گۈزەل (ماكانىم).

(قەلىب سەيناسى)ياشنايدۇ مەڭگۈلۈك،

(كەڭساي)دەك كۆركەمدۇر ئۇشبۇ بوستانىم.

(ئاقىللار ) دائىملا  دەيدىغۇ  شۇنداق،

ئەي ئوغلۇم(ئىز)قالدۇر تارىخ بىتىدە.

(بىلىملەر)ئۈگەنگىن ھەم ئالغا(باس باس)،

(پاكنۇر)بۇپ چاقنىسۇن روھىڭ كىچىدە.

(مىسرانىم ) چىللايدۇ(  ئوتتۇز ئوغۇل)نى،

ياۋلارغا چىقار دەپ (مىسران ) قىلىچنى.

(روھلان)غان ھەر يۈرەك (ھۇررا) دەيدىغۇ،

(پاك چىن)دىن  تىكلىدى ئۇيغۇر (ئوبراز)نى.

ئەي (دوستۇم) يولىمىز بولسۇن (بىلىمزار )،

ھەر ( قەلىب سەيناسى )   ئاڭا   ئىنتىزار.

ئارمىنىم شۇ بولسام بىر ياخشى ( ئوغلان)،

(گۆھەردەك )(پاك زىمىن) ساڭا (ئالمىزار).

( زوقمەن )بۇپ قارايدۇ ئۆزگى ئەللەرمۇ،

(نۇرلان) غان قەلبىمىز مەڭگۈ (تۇتاش)سا.

(ساپ ھاۋا) ئىلىڭلار  كىلىپ  (تورداشلار)،

( ئانا دىيار)ىم شۇ ( 8 كوچا )غا.

                    مەنبە : دوستۇم ئوت يازمىسى
