مۇنبەر تەھرىرلىگۈچىدىكى تېكىست دېگەن تاللاشنى ئېلىۋەتسىڭىز، خۇددى word تا خەت يازغاندەك كۆرۈپ تۇرۇپ تەھرىرلىگىلى بولىدۇ. ئەمما بۇ خىل ھالەتتە  Firefox, Chrome لاردا نۇرمال ئۇيغۇرچە كىرمەيدۇ. بۈگۈن  Firefox, Chrome لاردىمۇ نورمال ئۇيغۇرچە كىرگۈزىدىغان قىلىپ چىقتىم. سىناقتا ئۆتسىلا رەسمىي ئىشلىتىشكە كىرىشتۈرسەك بولىدۇ.
شۇڭا سىناققا قاتنىشىپ، مەسىلە بولسا بايقىشىپ بىرىشىڭلارنى سورايمەن.
ئاۋۋالقى ئىزدىنىشلەر:
Iframe غا Firefox, Chrome لاردا بىۋاستە ئۇيغۇرچە كىرگۈزۈش ھەققىدە ئىزدىنىش(1)
Iframe غا Firefox, Chrome لاردا بىۋاستە ئۇيغۇرچە كىرگۈزۈش ھەققىدە ئىزدىنىش(2)
يۇقارقى ئىزدىنىشلەردىن كېيىن يەنە ئۇلاپ ئىزدىنىش ئارقىلىق يەنە ئازدۇر-كۆپتۇر نەتىجىگە ئېرىشتىم. Iframe غا(يەنى ھەر خىل كۆرۈنمە يۈزلۈك تەھرىرلىگۈچلەرگە) Firefox, Chrome لاردا بىۋاستە ئۇيغۇرچە كىرگۈزۈش ھەل بولدى. ئەمدىكى مەسىلە سىناش ۋە كود ئەلالاشتۇرۇشتا قالدى.
ئارتۇقچىلىقى:
ھازىر IE, Firefox, Chrome لاردا نورمال ئۇيغۇرچە كىرگۈزگىلى بولىدىغان ھالەتكە كەلدى.
Ctrl+K بېلەن ئۇيغۇرچە-لاتىنچە ئالماشتۇرۇش ئەمەلگە ئاشۇرۇلدى.
باشقا بەتلەرگە كىرىشتۈرۈپ ئىشلىتىش قولايلاشتۇرۇلۇپ، بىر ID، بىر فۇنكىسىيە قوشۇش ئارقىلىقلا ئەمەلگە ئاشىدىغان بولدى.
يېڭىدىن تەھرىرلىگۈچ قوشۇش ۋە تەھرىرلىگۈچكە ئۇيغۇرچە كىرگۈزۈش ئىقتىدارى ئايرىلدى.
تەييار تەھرىرلىگۈچكە پەقەت ئۇيغۇرچە كىرگۈزۈش ھالىتى قوشىدۇ، تەھرىرلىگۈچكە ئارتۇقچە چېقىلمايدۇ.
كۆرگەزمە:
ساپ HTML بەتتىكى كۆرگەزمە: كىرىڭ
Discuz!X2 غا كىرىشتۈرگەن كۆرگەزمە: كىرىڭ(نۇرقۇتتىكى ئىسمىڭىز بېلەن كىرىپ تېما يېزىش كۆرۈنمە يۈزىدە سىناڭ.)
(سىناق تاماملانغاندىن كېيىن رەسمىي ئىشلىتىشكە كىرىشتۈرىلىدۇ، شۇنداقلا ئىشلىتىش ئۇسۇلى تەمىنلىدۇ.)
پىكىر-تەكلىپ قوبۇل قىلىمىز:
بۇ Js نىڭ تېخىمۇ مۇكەممەللىشىشى ئۈچۈن پىكىر-تەكلىپ قوبۇل قىلىمىز. تېخنىكىسى بارلارنىڭ يىتەرسىزلىكنى تولۇقلاشقا ھەمدەمدە بولۇشىنى ئۈمىد قىلىمىز. مەسىلە ھەل بولدى. ئەمدى زۈرۈر ئورۇنلارنى ئەلالاشتۇرساقلا بولىدۇ.
Opera غا ئۇيغۇرچە كىرگۈزۈش ئەمەلگە ئاشۇرۇلمىدى. ۋاقت چىقىرىپ ئىزدەنسەك بولىدۇ.
Safari دا ئۇيغۇرچە كىرىدۇ، ئەمما ھەرپلەر باغلاشمىدى. ۋاقت چىقىرىپ ئىزدەنسەك بولىدۇ.
