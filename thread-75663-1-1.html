كۇنۇپكا تاختىسىدىكى ئەپچىل قوللىنىشلار

1. Ctrl + A ھەممىنى تاللاش

2.  Ctrl + B  ساقلىغۇچنى رەتلەش

3. Ctrl + C كۆچۈرۈش

4.Ctrl + O  كۆزنەكنى ئىچىش

5. Ctrl + F  ئىزدەش

6. Ctrl+ D   ھۆججەت يۇيۇش بۇيرۇقى

7. Ctrl + H   بۇرۇنقى ھۆججەتنى ئىچىش(ۋورد)

8. Ctrl + I  ساقلىغۇچنى ئىچىش

9. Ctrl + K ۋوردقا تور  ئادىرىسى قوشۇش

10. Ctrl + M  توربەتتە  شۇ بەتنى ساقلاش رولىنى ئۆتەيدۇ ، ۋوردتا تەڭشەك رولىنى ئۆتەيدۇ

11. Ctrl + N  يىڭى كۆزنەكنى ئىچىش

12. Ctrl + S  ساقلاش

13. Ctrl+U  ۋوردتا  تاللىغان مەزمۇننىڭ ئاستىغا سىزىقچە قوشۇش

14. Ctrl + V   چاپلاش

15. Ctrl + W   ئىشلىتىۋاتقان كۆزنەكنى تاقاش

16. Ctrl + X  كېسىش

17. Ctrl + Y  ئالدىنقى ھالەتكە قايتۇرۇش

18.Ctrl + Z  ئالدىنقى بۇيرۇقنى ئەمەلدىن قالدۇرۇش

19. F1   ياردەم

20. F2  قايتا ناملاش

21. F3   ئىزدەش بۇيرۇقى

22. F4  كۆرگۈچ ئادىرسىنى چىقىرىش (توربەت ، مېنىڭ كومپىيۇتۇرۇم قاتارلىقلاردا ئىشلەيدۇ)

23. F5  يىڭىلاش

24. Windows   مۇندەرىجىنى ئىچىش

25. Windows+D  ئىكرانغا قايتىش

26. WIN+ U  باشقۇرۇش كۆزنىكىنى ئىچىش

27. Windows+F   بارلىق ھۆججەتنى ئىزدەش

28. Windows+L   ئابونت يۆتكەش  

29.  Windows+M   كۆزنەكنى كىچىك ھالەتكە ئايلاندۇرۇش

30.Alt + Enter   ئىكراننى چوڭ ھالەتكە ئايلاندۇرۇش   

31. Alt + F4    ئىشلىتىۋاتقان كۆزنەكنى تاقاش ياكى كومپىيۇتۇرنى ئۆچۈرۈش

32. Alt + بوشلۇق    ئىشلىتىۋاتقان كۆزنەكنىڭ باشقۇرۇش سەھىپىسىنى ئىچىش     

33. Ctrl + Esc    ئىكران تىزىملىكىنى ئىچىش

34. Esc    كۆزنەكتىن چىكىنىش

35. WIN+ E     مېنىڭ كومپىيۇتۇرۇمنى ئىچىش

36.Ctrl+بوشلۇق       يىزىق ئالماشتۇرۇش
