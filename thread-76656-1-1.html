      4-ئاينىڭ 19-كۈنىدىكى ئۇچۇر، چەتئەل ئاخبارات ۋاستىلىرىنىڭ خەۋرىگە ئاساسلانغاندا ، مىكروسوفت Windows 8نىڭ نەشىرلىرىنى ئېلان قىلغان .  ئىشلەتكۈچىلەر بۇ سۇپىغا قارتا يەنە نىمىلەرنى بىلىشى كېرەك؟  مىكروسوفت بۇ قېتىم ئىستىمالچىلار، كىچىك كارخانىلار ، چوڭ كارخانىلار ۋە  ARMبىرتەرەپ قىلغۇچىسى ئىشلىتىلگەن ئۈسكۈنە ئىشلەتكۈچىلىرگە ئوخشىمىغان تۆت خىل نۇسخىدىكى  Windows 8نى تەمىنلەيدۇ.
   مىكروسوفىت 4-ئاينىڭ16-كۈنى ئىستىمالچىلار ۋە كارخانا ئىشلەتكۈچىلىرى بىلىشكە تىگىشلىك ئىشلارنى ئاشكارلىغان .مىكروسوفت بۇيىل ئاخىرىدا ئوخشىمىغان نۇسخىدىكى Windows 8 مەشخۇلات سېستىمىسىنى تارقىتىدىغان بولۇپ . مۆلچەردىكىدەك، مىكروسوفت ئىستىمالچىلار ۋە تېخنىكا ھەۋەسكارلىرىغا ئىككى خىل نۇسخىدىكى Windows 8 تەمىنلەيدۇ،ھەمدە نامى « Windows 8RT » بولغان ARMسۇپىسىغا ماسلاشتۇرۇلغان نەشىرنى ، يەنە پەقەت چوڭ كارخانىلارغىلا ماسلاشتۇرۇلغان  Windows 8 كارخانا نۇسخىسىمۇ بار.
  ھازىر، ھەرخىل نۇسخىدىكى Windows 8 ئېلان قىلىندى، ئىستىمالچىلار ۋە كارخانا ئىشلەتكۈچىلىرى ئۆزلىرىنىڭ ھەربىر خىل نۇسخىدا قايسى ئىقتىدارلاغا ئېرىشەلەيدىغانلىقىنى ، قايسى نۇسخىنىڭ ئۆزلىرىگە تېخىمۇ ماس كېلىدىغانلىقىنى بىلىشى كېرەك. تۆۋەندىكىسى بولسا ئىشلەتكۈچىلەرنىڭ Windows 8نىڭ تۆت خىل نۇسخىسى توغرىسىدا بىلىشكە تىگىشلىك ئىشلار .
1.Windows 8 بولسا بارلىق ئىستىمالچىلارنىڭ ھەقىقى ئېھتىياجى
  Windows 8 تەمىنلىگەن ئىقتىدار سانىغا ئاساسلانغاندا، ئىستىمالچىلار بۇ سېستىمىنىڭ مەخسۇس نۇسخىسىنى ئىشلىتىشىنىڭ ھاجىتى يوق. كىشىلەر Windows 8دە بىرقىسىم ئۆچەملىك ئىقتىدارلارنى ، مەسىلەن Windowsكۆپ ۋاستە قويغۇچىسى  ، Windows دۇكىنى ۋە IE 10 قاتارلىق ئىقتىدارلارنى ئىشلىتەلەيدۇ.
2.Windows 8نىڭ كەسپىي نۇسخىسى كىچىك كارخانىلاردا كەم بولسا بولمايدۇ
    ساتقۇچى كىچىك كارخانىلار Windows 8نىڭ باشلانغۇچ دەرىجىدىكىسىنى ئىشلىتىپ ۋاقىت ئىسراپ قىلماسلىقى كېرەك. ئەكسىچە، بۇ كىچىك كارخانىلار  Windows 8نىڭ كەسپىي نۇسخىلىرىنى سېتىۋېلىشى لازىم . بۇ نۇسخىدا ئۆلچەملىك مەشخۇلات سېستىمىسى تەمىنلىگەن ئىقتىدارلارنىڭ ھەممىسى بولۇشتىن سىرت ، يەنە بىر قىسىم ئاچقۇچلۇق ئىقتىدارلار قېتىلغان، مەسىلەن BitLocker، Client Hyper-V، ھۆججەت شىفىرلاش سېستىمىسى قاتارلىقلار.
3.  Windows 8 RTدا كەم بولغان ئىقتىدار
  Windows 8 RTبولسا ARM بىرتەرەپ قىلغۇچىلىرىغا ماسلاشتۇرۇلغان بولۇپ ،مىكروسوفت ئېكىلوگىيىلىك سېستىمىسىدىكى بىر ھالقىلىق مەھسۇلات. لېكىن ، RTنۇسخىدىكى بىرقىسىم ئىقتىدارلارنى ئىشلەتكۈچىلەر ياقتۇرماسلىقى مۇمكىن. مەسىلەن ، بۇ نۇسخىدا Windows كۆپ ۋاستە قويغۇچىسى يوق. بۇنىڭدىن باشقا RTنۇسخىدا مىكروسوفىتنىڭ ساقلاش بوشلۇقى ئىقتىدارى يوق(Storage Space). بۇ ئىقتىدار بولسا مەۋھۇملاشتۇرۇلغان قاتلام ئارقىلىق ساقلاش باشقۇرۇش ئىقتىدارى تەمىنلەيدىكەن. لېكىن Windows 8 RT نۇسخىسىدىكى ئەڭ چوڭ مەسىلە بولسا x86/64 يۇمشاق دېتاللىرىنى ۋە ئۈستەل كومپىيوتېر يۇمشاق دېتاللىرىنى قاچىلاشقا بولمايدىكەن. باشقا نۇسخىدىكى Windows 8لاردا x86/64 يۇمشاق دېتاللىرىنى نورمال قاچىلىغىلى بولىدىكەن .
4.Windows 8نىڭ كارخانا نۇسخىسى چوڭ كارخانىلارنىڭ ئىشلىتىشىگە ماس كېلىدۇ
  مىكروسوفت گەرچە Windows 8نىڭ كەسپىي نۇسخىسى توغرىسىدا تەپسىلى ئۇچۇر بەرمىگەن بولسىمۇ، لېكىن ، بۇ نۇسخىدا ئىستىمالچىلار نۇسخىسى تەمىنلىگەن ھەممە ئىقتىدارلار سەپلەنگەن. مىكروسوفتنىڭ كۆرسەتكىنىگە ئوخشاش ، Windows 8نىڭ كارخانا نۇسخىسى چوڭ كارخانىلارنىڭ بىردىنبىر تاللىشى بولۇپ ، بۇ نۇسخىغا Windows 8نىڭ كەسپى نۇسخىسى تەمىنلىگەن ئىقتىدارلاردىن باشقا يەنە بەزى ئىقتىدارلار تەمىنلەنگەن، مەسىلەن PCباشقۇرۇش ۋە مەۋھۇملاشتۇرۇش قاتارلىقلار.
5. دەرىجە كۆتۈرۈش مۇلازىمىتىنى
  كىشىلەرنىڭ پەرەز قىلغىنىغا ئوخشاش، مىكروسوفت ھازىرقى ئىشلەتكۈچىلەرنى يېڭى نۇسخىدىكى Windows 8نى سېتىۋېلىشقا قىستىمايدۇ. ئەكسىچە ، مىكروسوفتنىڭ ھازىرقى Windows 7باشلانغۇچ نۇسخىسى، ئائىلە ئاساسى نۇسخىسى، ئائىلە كۈچەيتىلگەن نۇسخىسى ئىشلەتكۈچىلىرى سېستىمىسىنى Windows 8ۋە Windows 8كەسپى نۇسخىسىغا كۆتۈرەلەيدۇ.  Windows 7كەسپى نۇسخىسى ۋە Windows 7بايراقدار نۇسخىسى ئىشلەتكۈچىلىرى پەقەت Windows 8نىڭ كەسپى نۇسخىسىغىلا كۆتۈرەلەيدىغان بولۇپ Windows RTنۇسخىسىغا كۆتۈرۈشكە بولمايدىكەن.
6. يىراقتىن كونتىرول قىلىش ئىقتىدارى ھەممە نۇسخىدا يوق
  تېخنىكا ھەۋەسكارلىرى ۋە كارخانا ئىشلەتكۈچىلىرىگە نىسبەتەن، يىراقتىن كونتىرول قىلىش ئىقتىدارى Windows 8 دىكى كىشىنى ئەڭ جەلىپ قىلىدىغان ئىقتىدارلارنىڭ بىرى. لېكىن ، بۇ ئىقتىدار ھەممە نۇسخىدا يوق. ئەمەلىيەتتە ، مىكروسوفت يىراقتىن كونتىرول قىلىش ئىقتىدارىنى Windows 8كەسپى نۇسخىسى ۋە كارخانا نۇسخىسىغىلا تەمىنلەيدىغانلىقىنى ئېيتقان بولۇپ، Windows 8ۋە Windows 8 RT نۇسخىسىدا بۇ ئىقتىدار يوق .
7. ئىنتايىن ئىخچام
  مىكروسوفتنىڭ ئىلگىرى تارقاتقان ھەممە نۇسخىدىكى Windows Vistaۋە Windows 7مەشخۇلات سېستىمىلىرى ئېسىڭىزدىمۇ ؟ ئىشلەتكۈچىنىڭ ھەرقايسى نۇسخىلارنى چۈشىنىشى ئىنتايىن تەس ئىدى. لېكىن ، Windows 8نىڭ پەقەت تۆت خىل  نۇسخىسلا بار بولۇپ ،ئىككى خىل نۇسخىلا  ئىستىمالچىلار ئۈچۈن تەمىنلەنگەن. مىكروسوفت ئىلگىركى ئۇسۇلىنى ئۆزگەرتىشى بەلكىم مىكروسوفتنىڭ كىرىمىنى ئاشۇرۇشقا پايدىسى بولىشى مۇمكىن.
8.Windows 8 RTنۇسخىسىغا Officeيۇمشاق دېتالى سەپلىنىدۇ( باشقا نۇسخىدا يوق)
  كىشىنى ھەيران قالدۇرىدىغىنى ، مىكروسوفت Windows 8 RTنۇسخىسىدا Officeيۇمشاق دېتالى (Word、Excel、PowerPoint،OneNote) سەپلىنىدۇ. باشقا Windows 8ۋە Windows 8كەسپى نۇسخىلىرىدا بۇ يۇمشاق دېتال سەپلەنمەيدۇ. مىكروسوفت گەرچە نىمشىقا Windows 8 RTنۇسخىسىغا Officeيۇمشاق دېتالىنى سەپلەيدىغانلىقى ۋە بۇنىڭ Officeيۇمشاق دېتالىدىن كىرىدىغان كىرىمگە تەسىر يەتكۈزۈشى  توغرىسىدا چۈشەنچە بەرمىگەن بولسىمۇ، لېكىن، بۇ RTنۇسخىنى ئىشلەتكەن ئىشلەتكۈچىلەرنى ئىنتايىن رازى قىلىدۇ، ھەمدە ئۇلار پۇل خەجلىمەيلا بۇ ئىشخانا يۈرۈشلۈك يۇمشاق دېتاللىرىنى ئىشلىتەلەيدۇ.
9. Windows 8 RTنۇسخىسى ئۈسكۈنىلەرگە ئالدىن قاچىلىنىدۇ
  گەرچە ئىشلەتكۈچىلەر Windows 8ۋە Windows 8كەسپى نۇسخىلىرىنى  سېتىۋېلىپ ئېھتىياجىغا ئاساسەن خالىغانچە ئۆزىنىڭ كومپىيوتىرىغا قاچىلاشقا بولىدۇ، لېكىن Windows 8 RTنۇسخىسىدا بۇنداق تاللاش چەكلىمىگە ئۇچرايدۇ. مىكروسوفتنىڭ ئىپادىلىشىچە ، بۇ نۇسخىنى پەقەت تاختا كومپىيوتېر ۋە باشقا ئۈسكۈنىلەرگە  ئالدىن قاچىلاپ ئىشلىتىشكە بولىدۇ.
10.مىكروسوفت تېخىچە باھاسى ۋە بازارغا سېلىش ۋاقتىنى ئېلان قىلمىدى
    ئىستىمالچىلار ئۈچۈن ئەڭ مۇھىم مەسىلە بولسا ، بىز تېخىچە ھەرقايسى نۇسخىلاردىكى Windows 8نىڭ باھاسى ۋە تارقىتىلىش ۋاقتىنى بىلمەيمىز. مىكروسوفت ھەر قايسى نۇسخىدىكى Windows 8نى ئېلان قىلغان بلوگدا ئىپادىلىشىچە، ئۇلار تېخىمۇ كۆپ ئۇچۇرلارنى ئاشكارلاشنى ئويلىشىۋېتىپتۇ

ئنسان ئۆزئۆمرىدىن ئۈكۈنمەيدىغان بولسۇن            ،ئىمان بىلەن ھايادىن ئايرىلغان ۋاقتىڭ تىرىك  تۇرۇپ ئۆلگەن ۋاقتىڭدۇر،
