خاتىرە كومپىيوتىردا wifiتارقىتىشنىڭ ئاددى ئۇسۇلى

تۆۋەندە قەدەم باسقۇچلىرىنى تونۇشتۇرۇپ ئۆتىمەن.

1. ۋىندوسنىڭ باشلاش رەسىمىگە بىرنى چىكىمىز.


مۇنداق ھالەتكە كېلىدۇ:

2.تۆۋەندىكى ئورۇنغا cmdدەپ يازىمىز.

بۇچاغدا ئېكران تۆۋەندىكى ھالەتتە بولىدۇ:

3.مائۇسنى بۇيەرگە ئەكىلىپ ئوڭنى چىكىپ،以管理员身份运行نى تاللايمىز.

بۇچاغدا ئېكران تۆۋەندىكى قارا ئېكران ھالىتىگە كېلىدۇ.

4. تۆۋەندىكى قۇرنى قارا ئېكرانغا چاپلايمىز.(ئېنتىرنى باسىمىز)
netsh wlan set hostednetwork mode=allow ssid=isim key=mahpinomur

كۆك رەڭلىك يەرنى ئۆزىڭىزنىڭ ئېھتىياجى بويىچە ئۆزگەرتسىڭىز بولىدۇ ئەلۋەتتە.

5.تۆۋەندىكى قۇرنى چاپلايمىز.(ئېنتىرنى باسىمىز)
netsh wlan start hostednetwork

تۈگىدى.
(ئەگەر تارقىتىشنى توختاتماقچى بولسىڭىز تۆۋەندىكى قۇرنى چاپلاڭ.ئېنتىرنى ئۇنۇتماڭ)
netsh wlan stop hostednetwork

ئىزاھات:يۇقارقى مەشخۇلات ئارقىلىق بىز isimنامىدىكى مەخپى نومۇرى mahpinomurبولغان wifiنى ئاچتۇق.4-باسقۇچتىكى كۆك رەڭلىك ئورۇننى ئۆزگەرتىش ئارقىلىق wifiغا ئۆزىمىز خالىغان ئىسىم ۋە مەخپى نومۇرنى قىلىۋالساق بولىدۇ.

بۇ ئۇسۇل بەك ئاددى سىزمۇ سىناپ كۆرۈڭ!
