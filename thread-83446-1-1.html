Windows 8 مەشغۇلات سىستېمىسى 26-ئۆكتەبىر رەسمىي تارقىتىلىدۇ
مىكروسوفت Windows بۆلۈمىنىڭ لېدىرى سىتىۋېن (Steven Sinofsky) چارشەنبە كۈنى شىركەتنىڭ يىللىق سېتىش يىغىنىدا، كېيىنكى ئەۋلاد مەشغۇلات سىستېمىسى – Windows 8 نىڭ 10-ئاينىڭ 26-كۈنى رەسمىي تارقىتىلىدىغانلىقىنى جاكارلىدى.
Windows XP, Windows VISTA ۋە Windows 7 نىڭ ئىشلەتكۈچىلىرى 40 دوللار تاپشۇرۇپ، Windows 8 گە دەرىجە ئۆستۈرسە بولىدىكەن. Windows 7 مەشغۇلات سىستېمىسى 2009-يىلى 10-ئايدا ئېلان قىلىنغان بولۇپ، ھازىرغا قەدەر 630 مىليون نۇسخا سېتىلغان.
مىكروسوفتنىڭ دېيىشىچە، Windows 8 مەشغۇلات سىستېمىسى كۆچمە ئۈسكۈنە ۋە كومپيۇتېر ئارىسىدىكى ئارىلىقنى تېخىمۇ كىچىكلىتىدىكەن، تارقىتىش مۇراسىمىدا نۇرغۇن شىركەتلەر Windows 8 قاچىلانغان قاتتىق دېتال مەھسۇلاتلىرىنى ئېلان قىلىدىكەن. بۈگۈن دەل 99 كۈن قالدى.
مىكروسوفت بۇ كۈندە يەنى 2012-يىلى 10-ئاينىڭ 26-كۈنى (جۈمە) ئۇيغۇرچىنى ئۆز ئىچىگە ئالغان 109 تىلدىكى Windows نى دۇنيادىكى 231 دۆلەت ۋە رايوندىكى بازارغا تەڭ تارقىتىدۇ. بۇ كۈن يەنە قۇربان ھېيت كۈنىدۇر.
خەۋەر مەنبەسى:
Microsoft Windows 8 coming on October 26
ئەمەلىيەتتە Windows دىن باشقا يەنە بىر قانچە مەھسۇلاتلارمۇ ئۇنىڭدىن بۇرۇن ئۇيغۇرچىلاشتۇرۇلغان ئىدى، بىر قىسىم يېڭىلىقلارنى بەزى تورداشلار ھازىردىن باشلاپ ھېس قىلىشقا باشلاپتۇ. بەلكىم بۇ كۈندە ئۇيغۇرچىلاشتۇرۇلغان تېخىمۇ كۆپ مىكروسوفت مەھسۇلاتلارنى كۆرەلەيسىز. Windows پەقەت ئۇنىڭ ئىچىدىكى بىرسى خالاس. بۇ يىل ھەقىقەتەنمۇ نۇرغۇن يېڭىلىق بار. يىل ئاخىرىدا كومپيۇتېر بازىرىمۇ
قايتىدىن جانلىنىشى مۇمكىن.
Microsoft Uighur نى ئارايۈزگە ئىشلەتمىدى، ھەممىنى قىلغان ئۆزىمىز، ئۆزىمىز خەت نۇسخىلىرىنى بەك چىرايلىق ياسىۋېتىپ، ئۇلار ياسىغان نۇسخا يارىماي، سەت كۆرۈنۈپ قالدى-دە. ھېلىمۇ ئارايۈز خەت نۇسخىسى Segoe UI غا ئۇيغۇرچە قوشۇلدى. بۇ مىكروسوفتنىڭ ھەممە مەھسۇلاتلىرى ئۈچۈن ئورتاق ئىشلىتىلىدۇ. شۇڭا، بۇنى ئىشلەتكەن ھەرقانداق ئۈسكۈنىلەردە ئۇيغۇرچە ئۇلاشمىدى دەيدىغان گەپ بولمايدۇ-ئەمدى. ھازىر سەت بولامدۇ؟ چىرايلىق بولامدۇ؟ قانداقلا بولمىسۇن بىر چىقسۇن، سېتىلسۇن، ئومۇملاشسۇن، ئاندىن بىر گەپ بولار.
نەچچە يىلدىن بويان قىلىنغان شۇنچە تىرىشچانلىقلارغا دىيارىمىزدىن 10 مىڭ يۈرۈش سېتىۋالىدىغان كىشىلىرىمىز چىقماسما ؟؟ ئەھۋال بۇنداق ئېغىرمۇ ئەمەستۇ ؟ مەكتەپ ، ئەدىلىيە ، بانكا ، كەسپى ئورۇن ، مەمۇرى ئورۇنلار ، دوختۇرخانا ، قۇراللىق ساقچى قىسىم،نىفىت قېزىش ، ئىستىراتخۇانىيە……. دىگەندەك نۇرغۇن ھۆكۈمەت ئورۇنلىرىدا 4000يۇەندىن مۇئاش ئېلىپ ئىشلەۋاتقانلار نۇرغۇن ئىدىغۇ….ئەجەبا نەچچە يۈز يۈەننى خەجلەپ نەچچە يىللىق ئەجىرنى سېتىۋالغان بولسا شۇ ئۆزى ئانا تىلىدا راھەتتە ئىشلىتەتتىغۇ….شۇ ئىككى قىتىم ئولتۇرۇشقا ئاز بارسىلا بولىدىغان ئىش ئىدى.ئەگەر ھەددىدىن زىيادە قىممەتلا بولمىسا مەن ئوقۇغۇچى جېنىمدا ئىككىنى سېتىۋىلىشنى قارار قىلىۋاتىمەن.
