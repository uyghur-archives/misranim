مىكروسوفىت شىركىتى Windows 8 نى ئېلان قىلغاندىن بېرى قاچىلاشقا ۋاقىت يەتمەي تاشلاپ قويغان ئىدىم. بۈگۈن قاچىلاپ كۆرۈپ بېقىش ئىستىكىدە توردىن ئىزدەپ http://www.win8mi.com/down/windows-8-release-preview.html  بۇ يەردىن X86 لىق خەنزۇچە نەشىردىكى Windows 8 نى چۈشۈرۈدۈم. قاچىلاش جەرياندا كومپىيۇتېرىمنى بۇزۇپ قۇيۇشتىن ئەنسىرەپ مەۋھۇم كومپيوتىر قۇرۇش دېتالى Oracle VM VirtualBox نى چۈشۈردۈم (چۈشۈرۈش ئادرىسى: http://pan.baidu.com/share/link?shareid=7845&uk=251927556  ) .
    بىرىنجى قەدەمدە:   VirtualBox نى كومپىيۇتېرغا قاچىلاپ (قاچىلاش جەريانى ئاددىي، پەقەت كېينكى قەدەمنى بېسىۋەرسىڭىز بۇلىدۇ)، يېڭىدىن بىر مەۋھۇم كومپىيۇتېردىن بىرنى قۇردۇم :

5 سائەت ئالدىدا چىقىرىش
چۈشۈرۈش (44.1 KB)
VirtualBox win8





ئىككىنجى قەدەمدە: يېڭىدىن قۇرغان مۇلازىمىتىغا نىسەتەن تەكشەش ئېلىپ باردىم. (قۇرغان مۇلازىمىتىر نامى ئۈستىگە مائۇس ئوڭ كونۇپكىسىنى چىكىپ تەڭشەش (设置) ناملىق تۈرنى ئاچىمىز )  
1) سېستىما (系统) نىڭ ئىچىدە  ئىككىنجى تۈردە PAE/NX نىڭ ئالدىغا بەلگە قۇيىمىز، ئۈچۈنجى تۈردە VT-x/AMD-V ۋە ئاستىدىكىگە بەلگە قۇيىمىز.
2) كۆرسىتىش (显示) نىڭ ئىچىدە 3D ۋە 2D ئىقتىدارىغا بەلگە قويىمىز ھەمدە، كۆرسىتىش كارتىسىنىڭ قىممىتىنى مۇۋاپىق تەڭشەيمىز. (مەن 128 قىلىپ تەڭشىسەم نۇرمال ئىشلەۋەردى)
3) ساقلاش (存储) نىڭ ئىچىدە نۇر دېسكا (光驱) تۈرىدىن سىز چۈشۈرىۋالغان Windows 8 نىڭ ISO ھۆججىتىنى كۆرسىتىپ بېرىمىز. ھەمدە مۇقۇملاشتۈرۈشنى بېسىپ چېكىنىمىز.

ئۈچۈنجى قەدەمدە: مەۋھۇم كومپىيوتېرنى تاللاپ قوزغىتىش (启动) باسساق كومپىيۇتېر قوزغىلىشقا باشلايدۇ.

ئەمدى رەسمىي قاچىلاشنى باشلىسىڭىز بۇلىدۇ. قاچىلاش جەريانداكومپىيۇتېرغاچىلىنىدىغان دېسكىنى كۆرسىتىپ بىرىسىز، قاچىلاش شەكلىنى تاللايسىز (ئىككىنجى تۈرنى تاللىسڭىز بۇلىدۇ)، ئاندىن ئىشلەتكۈچى نامى، كومپىيوتېر  نامى قاتارلىقلارنى تولدۇرۇم بەرسڭىز قاچىلىنىدۇ.
