ئىلقۇتتىن ئىلاۋە:

ھەممىمىز كۆڭۈل بۆلىۋاتقان Windows 8 مۇ ئېلان قىلىندى. بۇ مەشغۇلات سىستېمىسى ئۆزىدىكى ئۆزگىچە يېڭىلىقلار ۋە ئالاھىدە ياخشى بولغان ئىقتىدارلار ئارقىلىق كەڭ كومپيۇتېر ھەۋەسكارلىرىنى ئۆزىگە تارتىپ كەلدى. ئەينى ۋاقىتتا Windows 7 ئېلان قىلىنغاندا نۇرغۇن تورداشلار بىراقلا Windows XP دىن Windows 7 غا كۆچۈشنى خالىمىدى. چۈنكى بۇ يېڭىچە مەشغۇلات سىستېمىسىنىڭ Windows XP بولغان پەرقى زىيادە چوڭ بولغاچقا، كەڭ ھەۋەسكارلار دەماللىققا بۇ يېڭى سىستېما Windows 7 دا مەشغۇلات ئېلىپ بارالماي، گاڭگىراپ قالدى ۋە ئاخىرى كۆپ ساندىكى ھەۋەسكارلار سىستېمىسىنى يەنىلا Windows XP غا قايتۇرۇپ ئىشلەتتى. بۇ يەردە ھەۋەسكارلارنىڭ Windows 7 غا دەرھال كۆچەلمەسلىكى بۇ شاش ئاتنى قانداق مىنىشنى بىلەلمىدى. چۈنكى بۇ شاش ئاتنى قانداق كۆندۈرۈش، ئۆزىمىزگە قانداق خىزمەت قىلدۇرۇش ھەققىدە بىرەر جايدا سىستېمىلىق ماتېرىيال يوق ئىدى. شۇ ۋەجىدىن تا ھازىرغىچە ئارىمىزدا Windows 7 دىن ئىبارەت بۇ شاش ئاتنى مىنىپ باقمىغانلار نۇرغۇن. مانا ئەمدى نۇرغۇنلىرىمىز بۇ Windows 7 دىن ئىبارەت شاش ئاتنى مىنىشكە ئۈلگۈرمەي تۇرۇپ ئۇنىڭدىنمۇ چاپقۇر ئارغىماق ھېسابلانغان Windows 8 مەيدانغا چۈشتى. بۇ ئارغىماق ھەممىمىزنىڭ ئۇنى تېزدىن كۆندۈرۈپ ئۆزىمىزگە خىزمەت قىلدۇرۇشىمىزغا مۇھتاج. ھەممىمىز بىلىمىز، ئات ھەر قانچە ياخشى بولسىمۇ، ئۇنى مىنىشنى بىلمىسەك ئۇ ھالدا بىز ئۈچۈن ئادەتتىكى ئات بىلەن ئارغىماقنىڭ ھېچقانداق پەرقى بولمايدۇ. شۇڭا بلوگگىر تورداشلارنىڭ بۇ چاپقۇر ئارغىماقنى تېزدىن كۆندۈرۈپ، ھەقىقىي پەيزىنى سۈرۈشىگە پايدىلىق بولۇشىنى كۆزدە تۇرۇپ، بۈگۈندىن باشلاپ، مەزكۇر چاپقۇر ئارغىماققا ئائىت ئورگان تەرەپ تەمىنلىگەن «ئارغىماقنى كۆندۈرۈش قوللانمىسى (Windows 8 产品指南)» نى تەرجىمە قىلىپ، كەڭ تورداشلارغا سۇنۇشقا تۇتۇش قىلدى. تورداشلارنىڭ مۇۋاپىق تۈزىتىش پىكىرى بېرىشىنى ئۇمىد قىلىمەن.

———————————————————————–

ئەمىسە، تولا گەپنى ئاز قىلىپ، بۇ چاپقۇر ئارغىماقنى قانداق كۆندۈرۈش ھەققىدە ئاساسىي مەزمۇنغا ئۆتۈپ كېتەيلى!

Windows كۆركەم، تېز، راۋان بولۇپ، تور بېكىتىڭىز، كىشىلىك مۇناسىۋىتىڭىز، قوللىنىشچان ئەپلىرىڭىز شۇنداقلا تېخىمۇ كۆپ مەزمۇنلارنى ئۆزىگە مۇجەسسەملەپ، سىزنى «باشلاش»تىن ئېكرانىڭىزدا ئۆزىڭىز كۆڭۈل بۆلىدىغان بارلىق مەزمۇنلارنى كۆرۈش پۇرسىتىگە ئىگە قىلىدۇ. Windows 8 ۋە Windows RT ئىلغار بولغان قاتتىق دېتاللاردا يېڭىلىق يارىتىشقا كەڭ سۇپا تەمىنلەپ، يېڭى بىر ئەۋلاد تاختا كومپيۇتېرى، ئېلىپ يۈرۈشكە ئەپلىك كومپيۇتېر ۋە بىر گەۋدىلەشكەن PC قاتارلىقلارنىڭ مەيدانغا كېلىشىنى ئىلگىرى سۈردى. بۇ ئەسلىھەلەر سەزمە كونترول (触控)، مائۇس ۋە كۇنۇپكا تاختىسىنىڭ زور ئەۋزەللىكىدىن تولۇق پايدىلىنىپ، بۇ ئۈچ ئەسلىھەنى ھازىرقى ئەڭ تېز، ئەڭ ئىشەنچلىك بولۇش ئاساسىدا مۇكەممەل ماسلاشتۇرغان.


Windows خىزمەت ۋە كۆڭۈل ئېچىشنى تېخىمۇ ئازادىلەشتۈردى. Windows ۋە PC نىڭ تاشقى جەھەتتىكى ئەپچىللىكى، لايىھەلىنىشتىكى كۆركەملىكى ھەمدە يېڭىچە ماتېرىيال ئىشلىتىلگەنلىكى ئۈچۈن، ئىقتىدارى ۋە باتارىيەنى ئىشلىتىش ۋاقتى جەھەتتە يېڭى بىر پەللىگە كۆتۈرۈلدى. Windows 8 ئىشلەتكۈچىلەر ئۈچۈن كۆڭۈل ئېچىش ۋە قوللىنىشنى مەركەز قىلغان. سىز دوستلىرىڭىز بىلەن بىرگە ئويۇنلاردىن ھوزۇرلىنىش، سىن ھۆججىتى ۋە ناخشا چۈشۈرۈش شۇنداقلا ئۇلارنى تېلېۋىزوردا قويۇش قاتارلىقلاردىن ھوزۇرلىنالايسىز.


سىز Windows ئەپ بازىرىدىن Windows 8 ۋە Windows RT ئۈچۈن ئەپكە ئېرىشەلەيسىز. سىز ئەپ بازىرىدىن زور مىقداردىكى ئەپلەرنى بايقاپ، ئۆزگىچە ئالاھىدىلىككە ئىگە ئەپلەرنى كۆرەلەيسىز. ياكى ئۇلارنى چېكىپ زىيارەت قىلىپ ئەپلەرنى تېخىمۇ چوڭقۇرلاپ چۈشىنىش پۇرسىتىگە ئىگە بولالايسىز.


Windows ئۆزئارا باغلىنىشنى ئىشقا ئاشۇرغان بولۇپ، ئىچكى قىسىمغا ئورۇنلاشتۇرۇلغان كۆچمە بەلباغ ئىقتىدارى 3G ۋە 4G نى قوللايدۇ. كۆچمە ھالەتتە، PC ئاپتوماتىك ھالدا ئەتراپتىكى ئىشلىتىشكە بولىدىغان Wifi نۇقتىسىنى ئىزدەپ تاپالايدۇ، سىز ھەر قانداق جايدا بۇ كۆچمە بەلباغلارغا ئازادە ئۇلاپ، ھەر ۋاقىت خىزمىتىڭىزنى داۋاملاشتۇرالايسىز. شۇنداقلا ئۇرۇق-تۇققان، دوست-بۇرادەرلىرىڭىز بىلەن ئالاقىلىشەلەيسىز.


ئۆزگىچىلىكىڭىزنى نامايەن قىلالايسىز




«باشلاش» ئېكرانىدا ئۆزگىچىلىڭىزنى تولۇق نامايەن قىلالايسىز. ئەڭ دەسلەپ كۆرۈش پەردىڭىزگە بۆسۈپ كىرىدىغىنى جانلىق، كۆركەم بولغان «باشلاش» ئېكرانىدۇر. «باشلاش» ئېكرانىدىكى ھەر بىر بۆلەك مەلۇم بىر ئالاقىلاشقۇچى، قوللىنىش (ئەپ)، تور بېكەت، قويۇش تىزىملىكى شۇنىڭدەك سىز ئۈچۈن ناھايىتى مۇھىم بولغان مەزمۇنلار بىلەن باغلانغان. بۇ ھەرگىزمۇ تىنچ رەسىملىك ئادەتتىكى ئۈستەل يۈزى ئەمەس. ئەكسىچە، سىز: ھالەت يېڭىلاش، ھاۋارايىدىن مەلۇمات، تەۋسىيەلىك ھۆججەت، شۇنىڭدەك تېخىمۇ كۆپ مەزمۇنلارنى كۆرەلەيسىز. سىز ھەتتا بىر قوللىنىشنى ئېچىشتىن بۇرۇن ئۇنىڭ ھەرىكەتچان يېڭىلانغانلىقىنى كۆرەلەيسىز. «باشلاش» ئېكرانىغا قانچىلىك بۆلەكنى ئورۇنلاشتۇرغىڭىز كەلسە شۇنچىنى ئورۇنلاشتۇرالايسىز، سىز بۇ بۆلەكلەرنى تۈرگە ئايرىپ، رەتكە تۇرغۇزۇپ ۋە ئىسىم قويالايسىز. شۇڭا ئۇلار سىزنىڭ ئارزۇيىڭىز بويىچە گەۋدىلىنىدۇ.
