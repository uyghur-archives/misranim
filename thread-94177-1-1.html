مىكروسوفىت شىركىتى تەرىپىدىن ئىشلىنىپ بازارغا سېلىنغان windows 8 كومپيۇتېر مەشغۇلات سېستىمىسى مەخسۇس بىز ئۈچۈن ئۇيغۇرچە تىل بولىقى ئىشلەپ بىزگە بىر كاتتا سوۋغا تەغدىم قىلدى دىسەكمۇ بولىدۇ، گەرچە تور بېتىمىز كومپيۇتېر بىلىملىرىنى ئاساس قىلغان تور بەت بولمىسىمۇ ئۇيغۇرچە يېزىقتىكى windows 8 سېستىمىسىنى تەشۋىق قىلىپ قويۇش ئۈچۈن تېما بېتىدىكى 2 - ئىنكاس قەۋىتىگە ئېلان رەسىمى قىستۇرۇلدى ھەمدە مەزكۇر تېما يوللاندى .

مىكروسوفت شىركىتى Windows8نى رەسمىي تارقاتتى

(نۇرقۇت تورىدىن كۆچۈرۈلگەن مەزمۇن )



2012-يىلى 10-ئاينىڭ 25-كۈنى كەچتە مىكروسوفت شىركىتى نيويۇركتا يېڭى بىر ئەۋلاد مەشغۇلات سىستىمىسى Windows8نى رەسمىي ئېلان قىلدى ،شۇنىڭ بىلەن بىرلا ۋاقىتتا مىكروسوفت شىركىتىنىڭ ئورگان تور بېكىتىدە ئەسلىدىكى سىستىمىنى Windows8گە يۇقىرى كۆتۈرۈش بولىقى چۈشۈرۈش ھۆججىتى بىرلا ۋاقىتتا ئېلان قىلىندى. جۇڭگودىكى ئابۇنتلار 248يۈەن تۆلەش ئارقىلىق ئەسلىدىكى سىستىمىسىنى Windows 8 Pro (كەسپى نۇسخىسى)غا يۇقىرى كۆتۈرەلەيدۇ.

Windows 8مەشغۇلات سىستىمىسىغا نەشىرنى يۇقىرى كۆتۈرۈش بۇلىقىنىڭ پۈتۈن دۇنيادىكى باھاسى 39.99دوللار (جۇڭگو 248يۈەن) قىلىپ تەڭشەلگەن بولۇپ، نۆۋەتتە ئىشلىتىۋاتقان سىستىمىلاردىن Windows Vista ،Windows XP ياكى Windows 7 قاتارلىق مەشغۇلات سىستىمىلىرى ئىشلىتىۋاتقان ئابۇنتلار، Windows 8 نەشىر يۇقىرى كۆتۈرۈش ياردەمچىسى(Upgrade Assistant)دىن پايدىلىنىپ، توردا سىستىمىسىنى Windows 8مەشغۇلات سىستىمىسىغا ئۆزگەرتىۋالسا بۇلىدۇ. بۈگۈندىن باشلاپ 2013-يىلى 1-ئاينىڭ 31-كۈنىگىچە سېتىشنى تېزلىتىش ئۈچۈن Windows 8مەشغۇلات سىستىمىسىنىڭ باھاسى 69.99ئامېرىكا دوللىرى ( تەخمىنەن 437يۈەن خەلق پۇلىغا تەڭ) قىلىپ تەڭشەلدى.
2012-يىلى 6-ئاينىڭ 2-كۈنىدىن 2012-يىلى 1-ئاينىڭ 31-كۈنىگىچە بولغان ئارلىقتا Windows 7مەشغۇلات سىستىمىسىنى سېتىۋېلىپ كومپيۇتېرىغا قاچىلاپ ئىشلەتكەنلەر Windows Upgrade Offer نەشىرنى يۇقىرى كۆتۈرۈش ياردەمچىسى ئارقىلىق ،سىستىمىنى 14.99دوللار(خەلق پۇلى 98يۈەن)تۆلەپ Windows 8 مەشغۇلات سىستىمىسىغا ئالماشتۇرالايدۇ. 140دۆلەتنى قوللاش ئۈچۈن سىستىما نەشىر يۇقىرى كۆتۈرۈش ۋاقتى 2013-يىلى 2-ئاينىڭ 28-كۈنىگىچە بۇلىدىكەن.

جۇڭگو رايونىنىڭ ئالاھىدە ئەھۋالىنى كۆزدە تۇتۇپ، قاپلىق نۇسخىسىنى بازارغا سالمايدىكەن، ھازىرچە پەقەر توردىن سېتىۋېلىشقا بولىدۇ.

Windows 8 ئۇيغۇرچە تىل بولىقى رەسمىي ئېلان قىلىندى

( ئالىم ئەھەت بلوگىدىن كۆچۈرۈلگەن مەزمۇن )




ئەسسالامۇئەلەيكۇم، ھۆرمەتلىك تورداشلار:
ھەممىمىز ئىنتىزارلىق بىلەن كۈتكەن Windows 8 نىڭ ئۇيغۇرچە تىل بولىقى رەسمىي ئېلان قىلىندى. 3 كۈننىڭ ئالدىدا بۇ ئىشلارنى تېخى چۈشۈمدە ئېنىق كۆرگەن ئىدىم. مانا بۈگۈن 2012-يىل 12-ئاينىڭ 12-كۈنى رەسمىي ئېلان قىلىندى. ئەمدى بەزى ئىشلارنى ئۇلار رۇخسەت قىلغان دائىرە ئىچىدە بۇ يەردە رەسمىي ئاشكارىلاشقا بولىدۇ.

ئۇيغۇرچە تىل بولىقىدىكى بارلىق ھۆججەتلەرنىڭ ئاخىرقى تەرجىمىسى قۇربان ھېيتنىڭ ھارپا كۈنى پۈتۈنلەي تاماملانغان بولۇپ، ئۇنىڭدىن كېيىن مۇناسىۋەتلىك بەزى قوشۇمچە ماتېرىياللارنىڭ ئاخىرقى تەرجىمىسىنى قايتا تەكشۈرۈش، رەسمىي ئېلان قىلىشقا ئىمزا قويۇش قاتارلىق ئىشلار بىلەن مەشغۇل بولغان ئىدىم، 11-ئاينىڭ 30-كۈنى (جۈمە) ئۈرۈمچى ۋاقتى 6:00 دە ئەڭ ئاخىرقى خىزمەت پۈتۈنلەي ئاخىرلاشتى. ئەڭ ئاخىرقى ماتېرىياللارنى پىدا كىتابخانىسىدىكى (ئۇيغۇرسوفتنىڭ سابىق خىزمەتچىسى) مۇھەممەد توختى ۋە جىڭ ناھىيەسىدىكى ئوسمان (ئىزدىنىشتىكى ئۆردەك) ئاكاممۇ كۆرۈپ بەرگەن ئىدى. ئۇلارغا رەھمەت ئېيتىمەن. دېمەك بۇ خىزمەتكە 2 يىل 1 ھەپتە ۋاقىت كېتىپتۇ. مېنى قوللىغان بارلىق كىشىلەرگە ئالاھىدە رەھمەت ئېيتىمەن. ئەمەلىيەتتە بۇ ھەممەيلەننىڭ ئورتاق تۆھپىسىدۇر.

بايا يەنە ئاكامدىن كەلگەن تەبرىك تېلېفوننى، ئارقىدىنلا Microsoft تتىن كەلگەن تەبرىك تېلېفوننىمۇ قوبۇل قىلدىم. ئۇلار ئۇيغۇرچە تىل بولىقى ئۈچۈن ئىزچىل كۆڭۈل بۆلگەنلەر بولۇپ، Microsoft نىڭ بۇ ئىشقا مەسئۇل يۇقىرى قاتلام خادىمنىڭ دېيىشىچە بۇ ئىش پۈتۈنلەي Microsoft تەلەپ قىلغان سۈپەت بىلەن تاماملانغانلىقىنى، شىنجاڭ ئۇيغۇر ئاپتونۇم رايونىدا Windows 8 نىڭ سېتىلىش ئەھۋالى نىسبەتەن ياخشى ئىكەنلىكىنى ئۇقتۇردى. يەنە نۇرغۇن گەپلەر بار ئىدى، چۈشتىن كېيىن مەكتەپتە يىغىن بار ئىكەن. ۋاقتىم چىقسىلا، بۇ جەرياندىكى ئىشلارنى ئۇلار رۇخسەت قىلغان دائىرە ئىچىدە يېزىشقا تىرىشىمەن.

مەن نېمە ئويدا، سىلەر نېمە كويدا؟

( ئالىم ئەھەت بلوگىدىن كۆچۈرۈلگەن مەزمۇن )


Windows 8 نىڭ ئۇيغۇرچە تىل بولىقى ئورگان تەرەپتىن ئېلان قىلىنىپ ھەممەيلەن خۇشاللىققا چۆمگەن بۇ كۈنلەردە، ھەرقايسى تور بېكەتلەردىن تورداشلارنىڭ بۇ ھەقتە بەس-بەستە يازغان تېما ئىنكاسلىرىنى كۆرۈپ چىقتىم. ئەمما بەزى تور بېكەتلەرنى كۆرۈپ ھەيران قالدىم. مەن گەپنى سەل باشتىن باشلاي.

Windows 7 خەنزۇچىدىن باشقا يەنە جۇڭگودىكى 7 مىللەتنىڭ تىل-يېزىقىنى بىر تەرەپ قىلالايتتى. ئۇلار زاڭزۇچە (藏语)، مۇڭغۇلچە (蒙古语)، يىزۇچە (彝语)، ئۇيغۇرچە (维吾尔语)، ئۇنىڭدىن باشقا يەنە 德宏傣文, 西双版纳新傣文, 八思巴文 لاردىن ئىبارەت. Microsoft نىڭ ئۇيغۇرچە تىل بولىقىغا قانداق كۆڭۈل بۆلۈۋاتقانلىقىنى مەن بەش قولدەك ئوبدان بىلىمەن، ئورگان تەرەپ تىل بولىقىنى چىقىرىمەن دېسە، قائىدە بويىچە Windows قا كىرگەن تەرتىپ بويىچە ئويلىشىدۇ. ھېچبولمىغاندا جۇڭگودا خەنزۇچىدىن باشقا يەنە بىرەر تىلنى تاللايمەن دېسە زاڭزۇچە (تىبەت) ياكى مۇڭغۇلچىنى تاللىغان بولاتتى. زاڭزۇچە ۋە مۇڭغۇلچە يۇمشاق دېتاللارنىڭ تېخنىكىسىمۇ خېلە پىشىپ يېتىلگەن، بەزى دېتاللىرىنىڭ تېخنىكىسى تېخى بىزنىڭ ئالدىمىزدا. ئەلۋەتتە دۆلەتمۇ ئۇيغۇرچە يۇمشاق دېتاللارنىڭ تەرەققىياتى ئۈچۈن نۇرغۇن مەبلەغ سېلىۋاتىدۇ. ئەمما مەيلى قانداقلا بولمىسۇن بۇ قېتىم ئورگان تەرەپ ئۇيغۇر تىلىنى ئالدىدا ئويلىشىپ، ئىلتىماسنى تەستىقلىدى، بۇنىڭ سەۋەبىنى كېيىن دەپ بېرەي. لېكىن كېيىن چوقۇم زاڭزۇچە (藏语) ياكى مۇڭغۇلچە (蒙古语)نىڭ ئىچىدىن بىرىنى تاللايدۇ.

Microsoft نىڭ ئۇيغۇرچە تىل بولىقىنى بەلگىلەنگەن پىلاندىن بۇرۇن ئېلان قىلىۋېتىشىنى مەنمۇ بىلمىدىم. چۈنكى، تىل قوشۇشتىن پارسچە (ئىران)، ئۆزبېكچە (ئۆزبېكىستان)، قازاقچە (قازاقىستان)،…. تىل بولىقىنى ياكى ئەرەب يېزىقى ئاساسىدىكى باشقا تىل بولىقى كۇرد (تۈركىيە، كۇردىستان)، پوشتۇ (پاكىستان، پەنجاپ)،…نى چۈشۈرۈپ سىناپ بېقىڭلار. بۇلارنىڭ ئىچىدە ئەرەبچە ۋە ئۇيغۇرچە تىل بولىقىدىن باشقا ئەرەب يېزىقى ئاساسىدىكى تىل بولىقىنى تېخىچە ئىشلەتكىلى بولمايدۇ. بەلكىم ئۇلار تەييار بولمىغان بولۇشى مۇمكىن، لېكىن ئۇلارغىمۇ ئاز قالدى.

ئەمدى گېپىمىزنىڭ مۇھىم مەزمۇنىغا كەلسەك، بۇ يەردە بەزى تور بېكەت (مۇنبەر ياكى بىلوگلار)نى ئاشكارە تىلغا ئالغۇم كەلمىدى، ئەمما بۇ تور بېكەتلەر Windows 8 نىڭ ئۇيغۇرچە تىل بولىقىنى قايتىدىن پىششىقلاپ ئىشلەپ، بەزى ئۇچۇرلارنى قوشۇپ، ئېنىق ئېيتقاندا پېتىغداپ ياكى بۇلغاپ بولغاندىن كېيىن بەيدۇ تور دىسكىسىغا يوللاپ، ئۆز تور بېكىتىدىن چۈشۈرۈش ئۇلانمىسىنى تەمىنلەپتۇ، ھەتتا ئۆز تور بېكىتىنىڭ ئېلانى سۈپىتىدە بىلوگىمغا ئىنكاس يېزىشقىمۇ ئۈلگۈرۈپتۇ. مەن نېمە ئويدا، سىلەر زادى نېمە كويدا؟ ئەمما بۇنداق قىلىشنىڭ ياخشى بولمايدىغانلىقىنى ھېس قىلغان بەزى پىكىرى ئۆتكۈر تورداشلارمۇ بۇ ئىشقا ۋاقتىدا دىققەت قىلىپ، ئۇنداق قىلماسلىقنى تاپىلاپتۇ.

بۇنداق قىلمىش «Microsoft Windows 8 تىل ئارايۈزى بولىقىنى ئىشلىتىش ئىجازەتنامىسى»گە ئاشكارە خىلاپلىق قىلغانلىق بولۇپ، Microsoft بۇنداق قىلمىشنى ئەڭ ئېغىر كۆرىدۈ، ۋاقتى كەلگەندە قانۇنىي جاۋابكارلىقىنى سۈرۈشتە قىلالايدۇ. كېيىن تىل بولىقىنى توردىن چۈشۈرۈش تەمىنلەنگەندە، چۈشۈرۈشتىن ئىلگىرى «ئىشلىتىش ئىجازەتنامىسى»(ئۇيغۇرچە)نى توردىن كۆرەلەيسىز. ھەرقانداق ئادەمنىڭ تىل بولىقىنى ئۆز ئالدىغا قايتا پىششىقلاپ ئىشلەشكە، ئۆز ئېلانلىرىنى قوشۇشقا،…ئاشكارە تارقىتىشقا قەتئىي رۇخسەت قىلىنمايدۇ.

يەنە بىرسى بۇ كەسپىي ئەخلاق جەھەتتىمۇ قەبىھ قىلمىشتۇر. ناۋادا Microsoft مەھسۇلاتلىرىغا زىيادە ئامراق بولساڭلار، مەن ئالاقىلىشىپ بېرەي، بېيجىڭدىكى Microsoft ئاسىيا-تىنچ ئوكيان قۇرۇلۇش ئاكادېمىيەسىگە بېرىپ 1-2 ئايلىق پىراكتىكانت خىزمەتچى بولۇپ ئىشلەپ كېلىڭلار، كەسپىي ئەخلاق ۋە قىزغىنلىقنىڭ قانداق بولىدىغانلىقىنى ھېس قىلىپ كېلىڭلار. لېكىن، بۇنداق ئەخلاقسىزلىق قىلماڭلار. باشقىلار كۆرسە بۇلاردىمۇ بۇنداق ئىش بار ئىكەن ئەمەسمۇ دەپ ھەيران قالىدۇ. بۇ ھەرگىزمۇ ياخشى نام-ئاتاق ئەمەس.

يەنە مۇھىم بىرسى ھەممە ئادەم ئورگان تور بېكىتىدىن ئۇيغۇرچە تىل بولىقىنى چۈشۈرۈپ قاچىلىسا، ئۇيغۇر تىل يېزىقىغا بولغان ئېھتىياجنى ۋە قىزغىنلىقنى ئېنىق بىلەلەيدۇ. بۇ Microsoft نىڭ باشقا مۇھىم مەھسۇلاتلىرىغىمۇ ئۇيغۇرچە تىل بولىقىنى كېڭەيتىشكە بەلگىلىك پايدىسى بولىدۇ.

ئەگەر تور بېكىتىڭىز بولسا، ئازىراق ئېلان ئورنىدىن ۋاز كەچسىڭىزمۇ Windows 8 نى تەشۋىق قىلىپ قويۇڭ. مەن تېخى بارلىق ئۇيغۇرچە تور بېكەتلەردە Windows 8 توغرىسىدا بەس-بەستە تەشۋىقات بولامدىكىن دەپ ئويلاپتىمەن. گەرچە چۆچۈرىنى خام سانىغان بولساممۇ، ھازىرمۇ كېچىكمەيسىز، تور بېكىتىڭىزدە ھازىرغىچە ئۈزۈلدۈرمەي ئىشلىتىۋاتقان Microsoft نىڭ ئاشۇ كىچىككىنە eot تېخنىكىسىنىڭ يۈزىنى قىلىپ بولسىمۇ، 2-3 ئاي تەشۋىق قىلىپ قويۇڭ. سېتىۋېلىشنى خالايدىغانلار ئېتىبار مەزگىلىدىن پايدىلىنىپ سېتىۋالسۇن. ئىناۋەت كارتىسى سەۋەبىدىن سېتىۋالالمىغانلارغا ئىناۋەت كارتىسى بارلار ياردەملىشىپ ئېلىشىپ بەرسۇن. لېكىن ئىمكان بار پايدا ئېلىشنى مەقسەت قىلماڭ. چۈنكى دۇرۇس يۇمشاق دېتال سېتىۋالىمەن دەپ ئالدىڭىزغا كەلدى، بۇ ھەرگىزمۇ ئاسان ئەمەس. 250 يۈەن بەرسىمۇ 2 يۈەننى قايتۇرۇپ بېرىڭ. ئەجرىڭىزنى باشقا يەردىن تېپىڭ. بىرسىگە ياخشىلىق قىلسىڭىز، چوقۇم شۇ كىشى قايتۇرۇشى كېرەك دەيدىغان تامادا بولماڭ. ياخشىلىق ھامان ياخشىلىقتۇر، بەلكىم سىز ئويلاپ باقمىغان باشقا كىشىلەر تەرىپىدىن سىزگە قايتىشى مۇمكىن.

ياكى بۇ دېگەن تىجارەت دېسىڭىز، دوللارچى تونۇشلىرىڭىزغا دەپ قويۇڭ. مۇشۇ قارا سوغۇقتىمۇ 100 دوللار ئالماشتۇرسا، ئاران نەچچە پۇڭ ياكى نەچچە موچەن پايدا ئالىمەن دەپ بانكىنىڭ ئالدىدا ئولتۇرىدۇ. ماۋۇ 40 دوللارلىق نەرسىنى ئېلىپ ساتىمەن دېسە گەپ يوق 1-2 دوللار پايدا ئالالايدۇ. ئۇلاردا دوللار چىقىم قىلالايدىغان نۇرغۇن كارتا بار.

ئارتۇق ۋاقتىڭلار بولسا، باشقىلارنىڭ نەرسىسىنى بۇزىمەن دەپ يۈرمەي يېڭى نەرسىلەرنى ئۆگىنىڭلار، مەسىلەن: Windows 8 دۇكىنىغا ئەپ يېزىشنى مەشىق قىلىڭلار. شىركەت بولساڭلار Windows دۇكىنىغا تىزىملىتىشقا ياردەملىشىمەن. شەخس بولۇپ ئەپ يېزىپ قالساڭلار Windows دىكى ئۇيغۇرسوفت دۇكىنىدا ئېلان قىلىپ سېتىۋالساڭلار بولىدۇ.

تور بېكەتلىرىڭىزدە بىرنەچچە ئاي بولسىمۇ Windows 8 توغرىسىدا تەشۋىق قىلالامسىز-يوق؟ Microsoft بىزدىكى يۇمشاق دېتال قىزغىنلىقىنى بىر كۆرۈپ باقسۇن، قېنى مەرھەمەت.
