    ئەسسالامۇئەلەيكۇم، مۇنبەرداشلارنىڭ Windows 8 توغرىسىدىكى تېمىلارنى كۆرۈپمۇ يەنە بىر قىسىم مەسىلىلەردە چۈشىنىش ھاسىل قىلالماي، قايتا قايتا سوئال سورىغانلىقى ۋە ھەقىقەتەن Windows 8 نىڭ قانداق تەلەپنى ئورۇندىغاندا ئاندىن قانداق سېتىۋېلىنىپ قاچىلىنىدىغانلىقىنى تېخىمۇ يىغىنچاق بىر يەرگە ئەكىلىش ئۈچۈن، قۇربىمنىڭ يېتىشىچە تۆۋەندىكى ئۇچۇرلارنى يوللاپ ئولتۇرۇپتىمەن. كەڭ قېرىنداشلارغا ئازغىنە پايدىسى تېگىپ قالسا ئەجەپ ئەمەس. بۇمۇ ئۇيغۇر كومپيۇتېر ئۇستىلىرىنىڭ ئۇيغۇر يېزىقىمىز ئۈچۈن قوشىۋاتقان تۆھپىلىرىنى قەدىرلەش، ئىگە تەرەپكە ئۇيغۇرلار ئىھتىياجىنى ھىس قىلدۇرۇش ئۈچۈن، Windows 8 نى كۆپ سېتىۋېلىش ئارقىلىق بۇندىن كېيىنكى قەدەملەرگە ئاساس سېلىش قاتارلىقلاردا ھەسسە بولۇپ قالغۇسى دېگەن ئۈمىدتىمەن.

Windows 8 نىڭ سىستېمىغا بولغان تەلىپى:
   ئەگەر Windows 8 كەسپى نەشىرى دەرىجە ئۆستۈرۈش بولىقىنى قاچىلىماقچى بولسىڭىز، كومپيۇتېر سىستېمىڭىز Windows XP SP3 ، Windows 7 لاردىن يۇقىرى بولۇشى كېرەك. سىستېمىڭىز Windows XP ياكى Windows Vista بولسا، قوللىنىشچان دىتاللارنى قايتا قاچىلىشڭىز كېرەك.
كونكىرىت تۆۋەندىكى تەلەپلەر ئورۇنلىنىشى كېرەك.
بىر تەرەپ قىلغۇچ: 1GHz ياكى ئۇنىڭدىن تېز .
RAM(ئىچكى ساقلىغۇچ) :  1GB 32位 ياكى  2GB 64位
قاتتىق دىسكا بوشلۇقى: 16GB ياكى 20GB .(ئشلىتىشكە بولىدىغان)
图形卡گىرافىك كارتىسى:  WDDM  قوزغاتقۇچ پروگرامما بار بولغان Microsoft DirectX 9 گىرافىك ئۈسكىنە.
ئىكران :  1366  768x

     كومپيۇتېرىڭىز يۇقىرىدىكى ئاساسى تەلەپلەرگە يەتكەن بولسا، تۆۋەندىكى ئادرىسقا كىرىپ < پەقەت 248 يۈەنگە كەسپى نەشىرىنى چۈشۈرگىلى بولىدۇ> دېگەن خەتنى بېسىپ سېتىۋالسىڭىز بولىدىكەن.
  سېتىۋېلىشتا قويغان تەلىپى: ھىچ بولمىغاندا سىزدە ئىناۋەت كارتىسى 信用卡 ( قەرز ئورنىدا ئالدىن پۇل چىقىرىپ خەجلىگىلى ۋە مال سېتىۋالغىلى بولىدىغان كارتا)  بولۇشى كېرەك، ئۇنداق بولمىغاندا ئادەتتىكى بانكا كارتىسىدىن پۇل چىقىم قىلىپ سېتىۋالغىلى بولمايدىكەن. مۇنبەردىكى ۋە باشقا تور بەتلەردىكى خالىس ياردەم قىلغۇچى قېرىنداشلار بىلەن ئالاقە قىلىپ ئۇلارنىڭ خاس كارتىلىرى ئارقىلىق سېتىۋالساقمۇ بولىدىكەن.
    بۇ مەسىلە ھەققىدە تەجىرىبىلىك ، كومپيۇتېر ئۇستىسى قېرىنداشلارنىڭ يۇقىرىدىكى بايانلاردا خاتا بولغان يەرلىرى بولسا ۋاقتىدا تۈزىتىش بېرىپ، كەڭ دوستلارنىڭ توغرا ئۇچۇردىن پايدىلىنىش ئۈچۈن ئاۋارە بولۇپ قويىشىڭلارنى ئۈمىد قىلىمەن !
