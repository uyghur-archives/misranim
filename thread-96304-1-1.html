LibreOffice ناھايىتى قالتىس پۈتۈنلەي ھەقسىز ئىشخانا يۇمشاق دېتالى.  LibreOffice نى لىنۇكس، Windows،Mac OS … قاتارلىق نۇرغۇن مەشغۇلات سىستېمىسىنى قوللايدۇ.          سەينادىكى LibreOffice غا مۇناسىۋەتلىك تېمىلارنى كۆرۈپ بېقىڭ.  Google دىن ئىزدەپ بېقىڭ

مەن ھازىر libreoffice دىكى ئېلېكترونلۇق جەدۋەل-  excel دا فۇنكىتسىيە ۋە فورمۇلا كىرگۈزۈپ سىناپ بېقىۋاتىمەن.
libreoffice excel دە تېز سۈرئەتتە مەشغۇلات قىلىش بۇيرۇقلىرى:
Ctrl+Shift+$        كەسىر سان ئىككى خانىلىق پۇل پىچىمى
Ctrl+Shift+%        كەسىر سانسىز پىرسەنت پىچىمى
Ctrl+Shift+!        كەسىر سان ئىككى خانە، - بەلگىسى  ۋە مىڭلىق ئايرىمىدىكى سان پىچىمى  
Ctrl+Shift+:        ھازىرقى ۋاقىت
Ctrl+;                 ھازىرقى چىسلا
Ctrl+Shift+ +         قىستۇرۇش ئەسكەرتمە كۆزنىكى
Ctrl+ -                       ئۆچۈرۈش ئەسكەرتمە كۆزنىكى
Ctrl+1        كاتەك پىچىمىنى كۆرسىتىش
Ctrl+A        پۈتۈن جەدىۋەل
Ctrl+B        توملاش پىچىمىنى قوللىنىش\ بىكار قىلىش  
Ctrl+C        كاتەك كۆچۈرۈش
Ctrl+D        مەزمۇن ئۆچۈرۈش
Ctrl+F         ئىزدەش- ئالماشتۇرۇش كۆزنىكى
Shift+F4 جەدۋەلگە نام قويۇش
Ctrl+U        ئاستى سىزىقنى قوللىنىش\ بىكار قىلىش
Ctrl+V         چاپلاش
Ctrl+W        دەپتەرنى تاقاش
Ctrl+X         كىسىش
Ctrl+Z        ئاخىرقى مەشغۇلاتنى بىكار قىلىش

3- ئىقتىدار كۇنۇپكىسى
F1        ياردەم
F2        ئاكتىپ كاتەكتىكى مەزمۇننىڭ ئاخىرى
Ctrl+F2 فۇنكىتسىيە يېتەكچىسى
F4      سانلىق مەلۇمات مەنبەسى
Ctrl+F4 دەپتەرنى تاقاش
Alt+F4  مەنىسى Excel نى تاقاش

Ctrl+F10    ئۇسلۇب كاتالوگى
F11     ئۇسلۇب ۋە پىچىم
Shift+F11          قېلىپ
Alt+F11  ماكرو كۆزنىكى
F12        گۇرۇپپا

Excel دا دا مەلۇم كاتەككە  √ نى كىرگۈزمەكچى بولسىڭىز Alt كۇنۇپكىسىنى بېسىپ تۇرۇپ كىچىك كۇنۇپكا تاختىسىدىن 41420 نى كىرگۈزۈپ  Alt كۇنۇپكىسىنى  قويۇپ بېرىڭ . بولامدىكەن؟
Alt كۇنۇپكىسىنى بېسىپ تۇرۇپ كىچىك كۇنۇپكا تاختىسىدىن 41409 نى كىرگۈزۈپ  Alt كۇنۇپكىسىنى قويۇپ بەرسىڭىز كۆپەيتىش بەلگىسى چىقىدۇ.
Alt كۇنۇپكىسىنى بېسىپ تۇرۇپ كىچىك كۇنۇپكا تاختىسىدىن 178 نى كىرگۈزۈپ  Alt كۇنۇپكىسىنى قويۇپ بەرسىڭىز كۇۋادرات بەلگىسى چىقىدۇ.
  Alt كۇنۇپكىسىنى بېسىپ تۇرۇپ كىچىك كۇنۇپكا تاختىسىدىن 179 نى كىرگۈزۈپ  Alt كۇنۇپكىسىنى قويۇپ بەرسىڭىز كۇپ بەلگىسى چىقىدۇ.
Alt كۇنۇپكىسىنى بېسىپ تۇرۇپ كىچىك كۇنۇپكا تاختىسىدىن 189 نى كىرگۈزۈپ  Alt كۇنۇپكىسىنى  قويۇپ بەرسىڭىز 1\2  چىقىدۇ.
Alt كۇنۇپكىسىنى بېسىپ تۇرۇپ كىچىك كۇنۇپكا تاختىسىدىن 190 نى كىرگۈزۈپ  Alt كۇنۇپكىسىنى قويۇپ بەرسىڭىز 3\4  چىقىدۇ.
Alt كۇنۇپكىسىنى بېسىپ تۇرۇپ كىچىك كۇنۇپكا تاختىسىدىن 123 نى كىرگۈزۈپ  Alt كۇنۇپكىسىنى قويۇپ بەرسىڭىز چوڭ تىرناق } چىقىدۇ.
  Alt كۇنۇپكىسىنى بېسىپ تۇرۇپ كىچىك كۇنۇپكا تاختىسىدىن 125 نى كىرگۈزۈپ  Alt كۇنۇپكىسىنى قويۇپ بەرسىڭىز چوڭ تىرناق  { چىقىدۇ.
