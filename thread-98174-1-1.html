Zekr بولسا «قۇرئان كەرىم» سۈرە-ئايەتلىرىنى زىيارەت قىلىدىغان، تەتقىق قىلىدىغان، ئۆگىنىدىغان ۋە ئىزدەيدىغان ئۈستەل يۈزى پروگراممىسى. ئۇ بىر خىل ئوچۇق كودلۇق سىستېما بولۇپ Javaدىن پايدىلىنىپ يىزىلغان، Eclipseسۇپىسىدا SWT قۇرال ئامبىرىدىن پايدىلىنىپ تۈزۈلگەن.
  Windows(2K, XP, Vista, 7, 8) ,Linux, Mac قاتارلىق كۆپ خىل مەشغۇلات سىستېمىسىدا ئىجرا بولىدۇ. Zekr كىڭىيىشچانلىققا ئىگە بولۇپ كۆپ خىل ئارا يۈز، قۇرئان تەرجىمە ھۆججىتى ۋە قىرائىتىنى قوللايدۇ.
بۇ پروگرامما Java مۇھىتىدا ئىجرا بولغاچقا ئاۋۋال  http://java.com/en گە كىرىپ Java نى قاچىلاڭ.

Zekr نى قاچىلاش باسقۇچلىرى

1. Zekr پروگراممىسىنى قوزغىتىش



2.ئۇيغۇر تىلىنى تاللاپ قاچىلاشنى داۋاملاشتۇرىمىز.



3.قاچىلايدىغان ئورۇننى تاللايمىز.



4.قاچىلاۋاتىدۇ



5.قاچىلاپ بولغاندىن كىيىن قوزغالغاندىكى ئارا يۈزى



6.تەرجىمە قوشۇش



7.قوشقان تەرجىمە ھۆججىتىنى تاللاش



8.«قۇرئان كەرىم» ئۇيغۇرچە تەرجىمىسى



9.قىرائەت ھۆججىتى قوشۇش



10.خەت نۇسخىسىنى تەڭشەش
